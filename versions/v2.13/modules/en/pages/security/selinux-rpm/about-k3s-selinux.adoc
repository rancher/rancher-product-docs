`k3s-selinux` provides policies for K3s. It is installed automatically when the K3s installer script detects that it is running on an RPM-based distro.

The `k3s-selinux` GitHub repository is https://github.com/rancher/k3s-selinux[here].

For more information about installing K3s on SELinux-enabled hosts, see the {url-k3s-advanced-selinux-support}[K3s documentation].
