== 当一个功能被标记为弃用我可以得到什么样的预期？

当功能被标记为"`已弃用`"时，它依然可用并得到支持，允许按照常规的流程进行升级。一旦升级完成，用户/管理员应开始计划在升级到标记为已移除的版本之前放弃使用已弃用的功能。对于新的部署，建议不要使用已弃用的功能。
