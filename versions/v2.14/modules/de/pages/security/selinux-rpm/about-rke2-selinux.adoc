`rke2-selinux` bietet Richtlinien für RKE2. Es wird automatisch installiert, wenn das RKE2-Installationsskript erkennt, dass es auf einer RPM-basierten Distribution läuft.

Das `rke2-selinux` GitHub-Repository ist https://github.com/rancher/rke2-selinux[hier.]

Für weitere Informationen zur Installation von RKE2 auf SELinux-aktivierten Hosts, siehe die https://documentation.suse.com/cloudnative/rke2/latest/en/install/methods.html#rpm[RKE2-Dokumentation.]
