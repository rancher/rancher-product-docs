`k3s-selinux` bietet Richtlinien für K3s. Es wird automatisch installiert, wenn das K3s-Installationsskript erkennt, dass es auf einer RPM-basierten Distribution läuft.

Das `k3s-selinux` GitHub-Repository ist https://github.com/rancher/k3s-selinux[hier].

Für weitere Informationen zur Installation von K3s auf SELinux-aktivierten Hosts siehe die https://documentation.suse.com/cloudnative/k3s/latest/en/advanced.html#_selinux_support[K3s-Dokumentation].
