Rancher verpflichtet sich, die Gemeinschaft über Sicherheitsprobleme in unseren Produkten zu informieren. Rancher wird Sicherheitswarnungen und CVEs (Common Vulnerabilities and Exposures) für Probleme veröffentlichen, die wir behoben haben. Neue Sicherheitswarnungen werden auch auf Ranchers GitHub https://github.com/rancher/rancher/security/advisories[Security-Seite] veröffentlicht.
