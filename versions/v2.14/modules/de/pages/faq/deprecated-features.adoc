[#_what_can_i_expect_when_a_feature_is_marked_for_deprecation]
== Was kann ich erwarten, wenn eine Funktion als ausgelaufen markiert wird?

In der Version, in der die Funktionalität als "ausgelaufen" markiert ist, wird sie weiterhin verfügbar und unterstützt sein, sodass Upgrades dem üblichen Verfahren folgen können. Nach dem Upgrade sollten Benutzer/Administratoren damit beginnen, die Nutzung der ausgelaufenen Funktionalität einzustellen, bevor sie auf die Version upgraden, die als entfernt markiert ist. Die Empfehlung für neue Implementierungen ist, die ausgelaufene Funktion nicht zu verwenden.
