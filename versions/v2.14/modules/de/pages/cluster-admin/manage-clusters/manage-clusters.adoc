[#_configuring_tools]
== Werkzeuge konfigurieren

Rancher enthält eine Vielzahl von Werkzeugen, die nicht in Kubernetes enthalten sind, um Ihre DevOps-Operationen zu unterstützen. Rancher kann mit externen Diensten integriert werden, um einen effizienteren Betrieb Ihrer Cluster zu ermöglichen. Werkzeuge werden in folgende Kategorien unterteilt:

* Warnmeldungen
* Notifier
* Protokollierung
* Überwachung
* Istio Service Mesh

Werkzeuge können über *Apps.* installiert werden.
