[IMPORTANT]
====
Die Rotation der Verschlüsselungsschlüssel ist standardmäßig aktiviert und kann nicht deaktiviert werden.
====

Um Schlüssel in der Rancher-GUI zu rotieren:

. Klicken Sie auf *☰ > Clusterverwaltung*.
. Wählen Sie *⋮ > Verschlüsselungsschlüssel rotieren* ganz rechts auf dem Bildschirm neben Ihrem gewählten Cluster aus:
+
image::rke2-encryption-key.png[Rotation der Verschlüsselungsschlüssel]

NOTE: Für weitere Informationen zur Verschlüsselungskonfiguration von RKE2-Secrets siehe bitte die https://documentation.suse.com/cloudnative/rke2/latest/en/security/secrets_encryption.html[RKE2-Dokumentation].
