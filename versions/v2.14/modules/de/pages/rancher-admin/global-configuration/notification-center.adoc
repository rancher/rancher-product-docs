Das Rancher-Dashboard liefert dynamische Benachrichtigungen und Systemwarnungen, um sicherzustellen, dass Sie über die neuesten Updates, Lebenszyklusereignisse und Statusinformationen zu Ihrer Implementierung informiert bleiben.

[#_dynamic_content]
== Dynamischer Inhalt

Rancher ruft dynamische Inhalte ab, einschließlich Benachrichtigungen über neue Versionen, End of Maintenance (EOM) oder End of Life (EOL) Status und andere wichtige Ankündigungen.

Dynamische Inhalte werden den Benutzern über drei Bereiche im Rancher-Dashboard bereitgestellt.

* *Benachrichtigungsbanner auf der Startseite*: Ein dynamisches Banner, das oben auf der Startseite für hochpriorisierte Ankündigungen angezeigt wird.
* *Bereich auf der Startseite unter den Links*: Ein dedizierter Bereich auf der Startseite, der sich unter dem *Links* Abschnitt befindet und relevante Updates und Ressourcen anzeigt.
* *<<_what_is_the_notification_center, Benachrichtigungszentrum>>*: Ein zentraler Knotenpunkt für alle Benachrichtigungen, einschließlich Systemwarnungen, Ressourcen und Ankündigungen.

Um mit diesen Benachrichtigungen zu interagieren, klicken Sie auf *Mehr erfahren* bei der aufgeführten Ressource. Sie können die Benachrichtigung auch schließen.

[#_what_is_the_notification_center]
== Was ist das Benachrichtigungszentrum?

Das Benachrichtigungszentrum fungiert als zentrales Repository sowohl für systemgenerierte Warnungen als auch für dynamische Ankündigungen. Sie können darauf zugreifen, indem Sie auf das Glockensymbol in der oberen rechten Ecke des Rancher-Dashboards klicken.

Benachrichtigungen werden nach Schweregrad und Typ kategorisiert:

* *Fehler* zeigt ein Problem mit hoher Schwere an.
* *Warnung* zeigt ein Anliegen mit mittlerer Schwere an.
* *Information* bietet allgemeine Updates mit niedrigerer Schwere.
* *Aufgabe* zeigt an, dass ein Prozess oder eine Handlung derzeit in Bearbeitung ist.
* *Erfolg* bestätigt, dass ein Prozess oder eine Handlung erfolgreich abgeschlossen wurde.

Zum Beispiel könnten Sie eine Benachrichtigung erhalten, wenn eine neue Version von Rancher verfügbar ist, oder einen direkten Link zu den Rancher-Release-Notizen.

Sie können ganz einfach durch Ihre Benachrichtigungen mit den Pfeiltasten nach oben und unten blättern. Bestimmte Benachrichtigungen können auch eine zugehörige Aktion enthalten, die es Ihnen ermöglicht, sofort zu reagieren.
