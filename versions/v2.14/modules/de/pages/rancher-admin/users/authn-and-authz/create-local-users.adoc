Die lokale Authentifizierung ist die Standardoption, bis Sie einen externen Authentifizierungsanbieter konfigurieren. Rancher speichert Benutzerkontoinformationen, wie Benutzernamen und Passwörter, lokal. Standardmäßig ist der `admin` Benutzer, der sich zum ersten Mal bei Rancher anmeldet, ein lokaler Benutzer.

[#_adding_local_users]
== Lokale Benutzer hinzufügen

Unabhängig davon, ob Sie externe Authentifizierung verwenden, sollten Sie einige lokale Authentifizierungsbenutzer erstellen, damit Sie Rancher weiterhin nutzen können, falls Ihr externer Authentifizierungsdienst auf Probleme stößt.

. Klicken Sie oben links auf *☰ > Benutzer & Authentifizierung*.
. Klicken Sie im linken Navigationsmenü auf *Benutzer*.
. Klicken Sie auf *Erstellen*.
. Füllen Sie das Formular *Benutzer hinzufügen* aus.
. Klicken Sie auf *Erstellen*.
