Innerhalb von Rancher hat jeder Benutzer eine Reihe von Einstellungen, die mit seinem Login verbunden sind: persönliche Präferenzen, API-Schlüssel usw. Sie können diese Einstellungen konfigurieren, indem Sie im Menü *Benutzereinstellungen* auswählen. Sie können dieses Menü öffnen, indem Sie auf Ihren Avatar klicken, das sich im Hauptmenü befindet.

image::user-settings.png[Benutzereinstellungen-Menü]

Die verfügbaren Benutzereinstellungen sind:

* {xref-user-api-keys}[API & Keys]: Wenn Sie programmgesteuert mit Rancher interagieren möchten, benötigen Sie einen API-Schlüssel. Befolgen Sie die Anweisungen in diesem Abschnitt, um einen Schlüssel zu erhalten.
* {xref-cloud-credentials}[Cloud-Anmeldeinformationen]: Verwalten Sie die von Maschinenpools verwendeten Cloud-Anmeldeinformationen, um {xref-launch-k8s-with-rancher}[Knoten für Cluster bereitzustellen].
* {xref-user-preferences}[Einstellungen]: Legt oberflächliche Präferenzen für die Rancher-Benutzeroberfläche fest.
* Abmeldung: Beendet Ihre Benutzersitzung.
