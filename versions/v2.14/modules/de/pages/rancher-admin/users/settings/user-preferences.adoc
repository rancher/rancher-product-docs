Sie können Einstellungen anpassen, um Ihr Rancher-Erlebnis zu personalisieren. So ändern Sie die Einstellungen:

. Klicken Sie auf Ihr Benutzeravatar in der oberen rechten Ecke.
. Klicken Sie auf *Einstellungen*.

[#_language]
== Sprache

Wählen Sie Ihre Sprache für die Rancher-Benutzeroberfläche. Die gültigen Optionen sind:

* Englisch
* 简体中文 (Vereinfachtes Chinesisch)

[#_theme]
== Theme

Wählen Sie Ihre Hintergrundfarbe für die Rancher-Benutzeroberfläche. Wenn Sie *Auto* wählen, ändert sich die Hintergrundfarbe um 18 Uhr von hell zu dunkel und wechselt um 6 Uhr wieder zurück.

[#_login_landing_page]
== Login-Startseite

Wählen Sie die Seite, die nach dem Anmelden angezeigt wird. Die gültigen Optionen sind:

* Die Startseite.
* Der Bereich, den Sie zuletzt besucht haben.
* Ein bestimmter Cluster Ihrer Wahl.

[#_display_settings]
== Anzeigeeinstellungen

Wählen Sie, wie bestimmte Informationen angezeigt werden sollen:

* Datumsformat
* Zeitformat
* Tabellenzeilen pro Seite
* Anzahl der Cluster, die im Seitenmenü angezeigt werden sollen

[#_confirmation_setting]
== Bestätigungseinstellung

Wählen Sie, ob eine Bestätigung angefordert werden soll, wenn Knotenpools verkleinert werden.

[#_advanced_features]
== Erweiterte Funktionen

* Aktivieren Sie "In API anzeigen".
* Zeigen Sie System-Namespaces, die von Rancher verwaltet werden (nicht zum Bearbeiten oder Löschen gedacht).
* Aktivieren Sie die Tastenkombination zum Umschalten zwischen Dunkel-/Hellmodus (Shift+T).
* Alle Typbeschreibungen ausblenden.
* Aktivieren Sie Entwicklerfunktionen für Erweiterungen.

[#_helm_charts]
== Helm-Charts

Wählen Sie, ob nur veröffentlichte Helm-Charts angezeigt werden sollen oder ob auch Vorabversionen einbezogen werden sollen. Eine Version wird als Vorabversion betrachtet, wenn sie der https://semver.org/#spec-item-9[Spezifikation] entspricht, die von https://semver.org/[Semantic Versioning 2.0.0] definiert ist. Ein Helm-Chart mit einer Version von `0.1.3-dev.12ab4f` erfordert, dass `Include Prerelease Versions` ausgewählt wird, um angezeigt zu werden.
