In vielen Unternehmensumgebungen haben Server oder VMs, die vor Ort betrieben werden, keinen direkten Internetzugang, müssen jedoch aus Sicherheitsgründen über einen HTTP(S)-Proxy mit externen Diensten verbunden werden. Dieses Tutorial zeigt Schritt für Schritt, wie man eine hochverfügbare Rancher-Installation in einer solchen Umgebung einrichtet.

Alternativ ist es auch möglich, Rancher vollständig Air-Gapped ohne Internetzugang einzurichten. Dieser Prozess wird ausführlich in der {xref-air-gap}[Rancher-Dokumentation] beschrieben.
