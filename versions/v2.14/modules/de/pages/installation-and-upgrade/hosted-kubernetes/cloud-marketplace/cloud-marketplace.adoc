Rancher bietet eine Integration mit Cloud-Marktplätzen, um Unterstützung für Installationen, die bei bestimmten Cloud-Anbietern gehostet werden, einfach zu erwerben. Darüber hinaus bietet diese Integration auch die Möglichkeit, ein Supportkonfigurationspaket zu erstellen, das dem Rancher-Support zur Verfügung gestellt werden kann.

Diese Integration unterstützt nur AWS.
