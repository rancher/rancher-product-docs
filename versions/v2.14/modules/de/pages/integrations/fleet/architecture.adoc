Fleet kann Implementierungen aus Git verwalten, die auf rohem Kubernetes-YAML, Helm-Charts, Kustomize oder einer beliebigen Kombination davon basieren. Unabhängig von der Quelle werden alle Ressourcen dynamisch in Helm-Charts umgewandelt, und Helm wird als Engine verwendet, um alles im Cluster bereitzustellen. Dies gibt Ihnen ein hohes Maß an Kontrolle, Konsistenz und Nachvollziehbarkeit. Fleet konzentriert sich nicht nur auf die Skalierbarkeit, sondern auch darauf, dem Nutzer ein hohes Maß an Kontrolle und Sichtbarkeit darüber zu verleihen, was genau im Cluster installiert ist.
