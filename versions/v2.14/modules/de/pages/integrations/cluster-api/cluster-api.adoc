https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}] ist ein https://kubernetes.io/docs/concepts/extend-kubernetes/operator/#operators-in-kubernetes[Kubernetes Operator], der den Lebenszyklus von bereitgestellten Kubernetes-Clustern verwaltet, indem er die Integration zwischen Ihrer Cluster API (CAPI) und Rancher bereitstellt. Mit Rancher Turtles können Sie:

* CAPI-Cluster in Rancher importieren, indem Sie den Rancher Cluster Agent in CAPI-bereitgestellten Clustern installieren.
* Verwalten Sie den https://cluster-api-operator.sigs.k8s.io/[CAPI Operator] mit der {url-turtles-capiprovider}[CAPIProvider]-Ressource.

Der Abschnitt {xref-turtles-overview}[Übersicht] beschreibt die Installationsoptionen, die Architektur von Rancher Turtles und eine kurze Demo. Für weitere Details siehe die https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}Dokumentation].
