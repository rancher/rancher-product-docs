Monitoring V2 kann auf einem Windows-Cluster bereitgestellt werden, um Metriken von Windows-Knoten mit https://github.com/prometheus-community/windows_exporter[prometheus-community/windows_exporter] (früher `wmi_exporter`) zu erfassen.

[#_cluster_requirements]
== Cluster-Anforderungen

Monitoring V2 für Windows kann nur Metriken von Windows-Hosts erfassen, die mindestens die Version `wins` v0.1.0 haben.  Um Monitoring V2 für Windows vollständig bereitstellen zu können, müssen alle Ihre Hosts diese Anforderung erfüllen.

[#_upgrading_existing_clusters_to_wins_v0_1_0]
=== Upgrade bestehender Cluster auf wins v0.1.0

Wenn der Cluster vor Rancher 2.5.8 bereitgestellt wurde (auch wenn die aktuelle Rancher-Version 2.5.8 ist), können Sie Monitoring V2 für Windows nicht erfolgreich bereitstellen, bis Sie die wins-Version auf jedem Host auf mindestens v0.1.0 upgraden.

Um dieses Upgrade zu erleichtern, hat Rancher 2.5.8 ein brandneues Helm-Chart namens `rancher-wins-upgrader` veröffentlicht.

. Stellen Sie `rancher-wins-upgrader` mit dem folgenden Override bereit:
+
[,yaml]
----
 # Masquerading bootstraps the wins-upgrader installation via
 # a previously whitelisted process path since the normal install path,
 # c:\etc\rancher\wins\wins-upgrade.exe is not normally whitelisted.
 # In this case, we are using the previously whitelisted process
 # path used by Monitoring V1.
 masquerade:
   enabled: true
   as: c:\\etc\wmi-exporter\wmi-exporter.exe
----

. Sobald alle Ihre Hosts erfolgreich geupgradet wurden, stellen Sie bitte sicher, dass Sie das Helm-Chart erneut mit den Standardwerten bereitstellen, um Konflikte mit den folgenden Einstellungen zu vermeiden:
+
[,yaml]
----
 masquerade:
   enabled: false
----

*Ergebnis:* Die Hosts sind bereit für die Installation von Monitoring V2. Sie können wählen, das `rancher-wins-upgrader` Chart zu deinstallieren oder es in Ihrem Cluster zu belassen, um zukünftige Upgrades zu erleichtern.

Für weitere Informationen darüber, wie es verwendet werden kann, siehe bitte das https://github.com/rancher/wins/blob/master/charts/rancher-wins-upgrader/README.md[README.md] des Charts.
