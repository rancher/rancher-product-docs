Installieren Sie *Project Monitors* in jedem Projekt, in dem Sie das Projektmonitoring aktivieren möchten.

. Klicken Sie auf *☰ > Clusterverwaltung*.
. Gehen Sie auf der Seite *Clusters* zu dem Cluster, in dem Sie das Monitoring aktivieren möchten, und klicken Sie auf *Erkunden*.
. Klicken Sie auf der linken Navigationsleiste auf menu:Monitoring[Project Monitors]. Klicken Sie dann oben rechts auf *Erstellen*.
+
image::project-monitors.png[Project Monitors]

. Wählen Sie Ihr Projekt aus dem Dropdown-Menü aus und klicken Sie dann erneut auf *Erstellen*.
+
image::create-project-monitors.png[Project Monitors erstellen]
