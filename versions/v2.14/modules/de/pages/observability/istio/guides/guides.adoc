[WARNING]
====
https://github.com/rancher/charts/tree/release-v2.11/charts/rancher-istio[Rancher-Istio] ist seit Rancher v2.12.0 ausgelaufen; verwenden Sie die https://apps.rancher.io[SUSE Application Collection] Build von Istio für verbesserte Sicherheit (in SUSE Rancher Prime-Abonnements enthalten).
Detaillierte Informationen finden Sie in https://forums.suse.com/t/deprecation-of-rancher-istio/45043[dieser Ankündigung].
====

Dieser Abschnitt beschreibt, wie Sie Istio aktivieren und in Ihren Projekten verwenden können.

Wenn Sie Istio für das Verkehrsmanagement verwenden, müssen Sie externen Verkehr zum Cluster zulassen. In diesem Fall müssen Sie alle unten aufgeführten Schritte befolgen.

[#_prerequisites]
== Voraussetzungen

Dieser Leitfaden geht davon aus, dass Sie bereits {xref-install-upgrade}[Rancher installiert haben] und dass Sie bereits {xref-cluster-deployment}[ein separates Kubernetes-Cluster bereitgestellt haben], auf dem Sie Istio installieren werden.

Die Knoten in Ihrem Cluster müssen die {xref-istio-cpu-memory-allocations}[CPU- und Speicheranforderungen] erfüllen.

Die Workloads und Dienste, die Sie von Istio steuern lassen möchten, müssen https://istio.io/docs/setup/additional-setup/requirements/[Istios Anforderungen] erfüllen.

[#_install]
== Installieren

[TIP]
.Quick-Setup-Tipp:
====

Wenn Sie keinen externen Verkehr benötigen, um Istio zu erreichen, und Sie Istio nur für die Überwachung und Nachverfolgung des Verkehrs innerhalb des Clusters einrichten möchten, überspringen Sie die Schritte zum {xref-istio-gateway-setup}[Einrichten des Istio-Gateways] und {xref-istio-traffic-management-setup}[Einrichten der Istio-Komponenten für das Verkehrsmanagement].
====


. {xref-enable-istio-in-cluster}[Aktivieren Sie Istio im Cluster.]
. {xref-enable-istio-in-namespace}[Aktivieren Sie Istio in allen Namespaces, in denen Sie es verwenden möchten.]
. {xref-istio-sidecar}[Fügen Sie Deployments und Dienste hinzu, die den Istio-Sidecar injiziert haben.]
. {xref-istio-gateway-setup}[Richten Sie das Istio-Gateway ein.]
. {xref-istio-traffic-management-setup}[Richten Sie die Istio-Komponenten für das Verkehrsmanagement ein.]
. {xref-istio-generate-and-view-traffic}[Generieren Sie Verkehr und sehen Sie Istio in Aktion.]
