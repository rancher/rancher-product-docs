[#_the_kiali_traffic_graph]
== Das Kiali-Verkehrsdiagramm

Die Istio-Übersichtsseite bietet einen Link zum Kiali-Dashboard. Vom Kiali-Dashboard aus können Sie Diagramme für jeden Namespace anzeigen. Das Kiali-Diagramm bietet eine leistungsstarke Möglichkeit, die Topologie Ihres Istio-Service-Meshs zu visualisieren. Es zeigt Ihnen, welche Dienste miteinander kommunizieren.

[#_prerequisites]
== Voraussetzungen

Um sicherzustellen, dass der Verkehr im Diagramm angezeigt wird, stellen Sie sicher, dass Sie Prometheus im Cluster installiert haben. `Rancher-istio` installiert Kiali und konfiguriert es standardmäßig, um mit dem `rancher-monitoring` Chart zu arbeiten. Sie können `rancher-monitoring` verwenden oder Ihre eigene Überwachungslösung installieren.

Darüber hinaus verwendet Kiali für Istio-Installationen der Version `103.1.0+up1.19.6` und später einen Token für seine Authentifizierungsstrategie. Wenn Sie versuchen, den Token zu generieren oder abzurufen (z. B. für die Anmeldung), beachten Sie, dass der Name des Kiali-Dienstkontos in Rancher `kiali` ist. Für weitere Informationen siehe die https://kiali.io/docs/faq/authentication/[Kiali-Token-Authentifizierungs-FAQ].

Optional: Sie können konfigurieren, in welchen Namespaces die Datenerfassung erfolgt, indem Sie die in {xref-selectors-and-scrape-configurations}[Selectors & Scrape Configs] beschriebenen Helm-Chart-Optionen festlegen.

[#_traffic_visualization]
== Verkehrsvisualisierung

Um das Verkehrsdiagramm anzuzeigen, folgen Sie den folgenden Schritten:

. Klicken Sie im Cluster, in dem Istio installiert ist, auf *Istio* in der linken Navigationsleiste.
. Klicken Sie auf den *Kiali* Link.
. Klicken Sie im Seitenmenü auf *Diagramm*.
. Ändern Sie den Namespace im Dropdown-Menü *Namespace*, um den Verkehr für jeden Namespace anzuzeigen.

Wenn Sie die URL zur BookInfo-App mehrmals aktualisieren, sollten Sie in der Lage sein, grüne Pfeile im Kiali-Diagramm zu sehen, die den Verkehr zu `v1` und `v3` des `reviews`-Dienstes anzeigen. Das Bedienfeld auf der rechten Seite des Diagramms ermöglicht es Ihnen, Details zu konfigurieren, einschließlich der Anzahl der Minuten des aktuellsten Verkehrs, die im Diagramm angezeigt werden sollen.

Für zusätzliche Werkzeuge und Visualisierungen können Sie Grafana und die Prometheus-Dashboards von der Seite *Überwachung* *Übersicht* aufrufen.
