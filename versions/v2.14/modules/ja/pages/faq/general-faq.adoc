このFAQは、Rancher v2.xに関して最も頻繁に尋ねられる質問に答えるために設計された進行中の作業です。

よくある技術的な質問については、xref:faq/technical-items.adoc[技術FAQ]をご覧ください。

ifeval::["{build-type}" != "srfa"]
[#_is_it_possible_to_manage_azure_kubernetes_services_with_rancher_v2_x]
== Rancher v2.xでAzure Kubernetes Servicesを管理することは可能ですか？

はい。AKSで利用可能なRancherの機能や、{xref-manage-clusters}[AKSに関するドキュメント]については、{xref-rancher-on-aks}[クラスター管理]ガイドをご覧ください。

[#_does_rancher_support_windows]
== RancherはWindowsをサポートしていますか？

はい。RancherはWindows Server 1809コンテナをサポートしています。Windowsワーカーノードを持つクラスターのセットアップの方法の詳細については、{xref-use-windows-clusters}[Windows用のカスタムクラスターの構成]に関するセクションを参照してください。
endif::[]

[#_does_rancher_support_istio]
== RancherはIstioをサポートしていますか？

はい。Rancherは{xref-istio}[Istio]をサポートしています。

[#_will_rancher_v2_x_support_hashicorps_vault_for_storing_secrets]
== Rancher v2.xは、秘密情報を保存するためにHashicorpのVaultをサポートしますか？

秘密情報の管理は私たちのロードマップにありますが、まだ特定のリリースには割り当てていません。

[#_does_rancher_v2_x_support_rkt_containers_as_well]
== Rancher v2.xはRKTコンテナもサポートしていますか？

現時点では、Dockerのみをサポートしています。

[#_does_rancher_v2_x_support_calico_contiv_contrail_flannel_weave_net_etc_for_embedded_and_registered_kubernetes]
== Rancher v2.xは、組み込みおよび登録されたKubernetes用にCalico、Contiv、Contrail、Flannel、Weave netなどをサポートしていますか？

Rancherは、Kubernetesクラスター用に以下のCNIネットワークプロバイダーを提供しています：Canal、Flannel、Calico、Weave。 公式にサポートされている内容の詳細については、 https://rancher.com/support-maintenance-terms/[Rancherサポートマトリックス]を常に参照してください。

[#_are_you_planning_on_supporting_traefik_for_existing_setups]
== 既存のセットアップに対してTraefikをサポートする予定はありますか？

現在、埋め込みTraefikサポートを提供する計画はありませんが、負荷分散の方法を引き続き検討しています。

[#_can_i_import_openshift_kubernetes_clusters_into_v2_x]
== v2.xにOpenShift Kubernetesクラスターをインポートできますか？

私たちの目標は、任意のKubernetesクラスターを運用することです。したがって、Rancher v2.xはOpenShiftと互換性がありますが、まだテストは行っていません。

[#_is_longhorn_product_name_integrated_with_rancher]
== {longhorn-product-name}はRancherと統合されていますか？

はい。LonghornはRancher v2.5以降と統合されています。
