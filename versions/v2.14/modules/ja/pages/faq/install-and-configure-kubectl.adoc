`kubectl`はKubernetesクラスターに対してコマンドを実行するためのCLI（コマンドラインインターフェース）ユーティリティです。これはRancher 2.xの多くのメンテナンスおよび管理タスクに必要です。

[#_installation]
== インストール

お使いのオペレーティングシステムに対するインストールについては、 https://kubernetes.io/docs/tasks/tools/install-kubectl/[kubectlのインストール]を参照してください。

[#_configuration]
== 設定

RKE2/K3sでKubernetesクラスターを作成すると、Kubeconfigファイルは選択したディストリビューションに応じて`/etc/rancher/rke2/rke2.yaml`または`/etc/rancher/k3s/k3s.yaml`に保存されます。これらのファイルはKubernetesクラスターへのアクセスを構成するために使用されます。

`kubectl`を使用して接続性をテストし、ノードのリストを取得できるか確認してください。

----
kubectl get nodes
 NAME                          STATUS    ROLES                      AGE       VERSION
165.227.114.63                Ready     controlplane,etcd,worker   11m       v1.10.1
165.227.116.167               Ready     controlplane,etcd,worker   11m       v1.10.1
165.227.127.226               Ready     controlplane,etcd,worker   11m       v1.10.1
----
