[#_what_is_ranchers_deprecation_policy]
== Rancherの廃止ポリシーとは何ですか？

公式の廃止ポリシーは https://www.suse.com/support/rancher-prime/#Rancher-Prime-Deprecation-Policy[Rancher Prime 廃止ポリシー]に文書化されています。

[#_what_can_i_expect_when_a_feature_is_marked_for_deprecation]
== 機能が廃止としてマークされた場合、何を期待できますか？

機能が「廃止」としてマークされたリリースでは、引き続き利用可能でサポートされており、通常の手順に従ってアップグレードすることが可能です。アップグレード後、ユーザーおよび管理者は、機能が削除対象として指定されたリリースにアップグレードする前に、廃止機能からの移行計画を立て始めるべきです。新しいデプロイメントに対する推奨は、廃止機能を使用しないことです。
