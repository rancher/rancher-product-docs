Rancherは、さまざまなボリュームプラグインを使用して永続ストレージをサポートしています。ただし、これらのプラグインを使用して永続ストレージをワークロードにバインドする前に、ストレージ自体を構成する必要があります。これは、サービスプロバイダーからのクラウドベースのソリューションであっても、あなた自身が管理するオンプレミスのソリューションであっても同様です。

お客様の便宜のために、Rancherは、人気のあるいくつかのストレージ方式の構成方法に関するドキュメントを提供しています：
