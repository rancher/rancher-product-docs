[#_configuring_tools]
== ツールの設定

Rancherには、DevOps操作を支援するためにKubernetesに含まれていないさまざまなツールが含まれています。Rancherは、クラスタをより効率的に運用するために外部サービスと統合できます。ツールは次のカテゴリに分類されています：

* アラート
* ノーティファイア
* ログ記録
* 監視
* Istioサービスメッシュ

ツールは*アプリ*を通じてインストールできます。
