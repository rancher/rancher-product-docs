[IMPORTANT]
====
暗号化キーのローテーションはデフォルトで有効になっており、無効にすることはできません。
====

Rancher UIでキーをローテーションするには：

. *☰ > クラスター管理*をクリックします。
. 選択したクラスターの右端にある*⋮ > 暗号化キーのローテーション*を選択します：
+
image::rke2-encryption-key.png[暗号化キーのローテーション]

NOTE: RKE2のシークレット暗号化設定の詳細は、 https://documentation.suse.com/cloudnative/rke2/latest/en/security/secrets_encryption.html[RKE2 ドキュメント]をご覧ください。
