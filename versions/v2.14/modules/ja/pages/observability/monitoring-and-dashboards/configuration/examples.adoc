[#_prometheusrule]
== PrometheusRule

PrometheusRuleには、通常 https://prometheus.io/docs/prometheus/latest/configuration/recording_rules/[Prometheusルールファイル]に配置するアラートおよび記録ルールが含まれています。

より詳細なアプローチとして、Prometheusリソースの`ruleSelector`フィールドは、PrometheusRulesリソースに付けられたラベルに基づいて、どのPrometheusRulesをPrometheusに読み込むかを選択できます。

例として、 https://prometheus-operator.dev/docs/developer/alerting/[Prometheus Operatorのドキュメント]を参照してください。そこにはPrometheusRuleの例があります。
