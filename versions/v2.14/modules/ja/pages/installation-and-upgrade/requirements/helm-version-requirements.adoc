このセクションでは、Rancherを高可用性Kubernetesクラスターにインストールするために使用されるツールであるHelmの要件について説明します。

[#_identifying_the_proper_helm_v3_version]
== 適切なHelm v3バージョンの特定

使用しているKubernetesバージョン範囲と公式に互換性のあるHelm v3バージョンを、私たちの https://www.suse.com/suse-rancher/support-matrix/all-supported-versions[Rancherサポートマトリックス]から選択してください。

このルールを適用するには、2つの外部リソースを参照する必要があります：

* *Helmバージョンの互換性:*https://helm.sh/docs/topics/version_skew/[Helmバージョンサポートポリシー]を参照し、Rancherのマイナーバージョンに対するルールに一致するバージョンを選択してください。
* *RancherのKubernetesサポート範囲:*https://www.suse.com/suse-rancher/support-matrix/all-supported-versions[Rancherサポートマトリックス]を使用して、ターゲットのRancherマイナーバージョンでサポートされているKubernetesバージョンを特定してください。

[#_example]
=== 例

* *場面:*ターゲットはRancher v2.11.4で、Kubernetesバージョン1.30から1.32をサポートしています。
* *アプリケーション:*この範囲をサポートするHelmバージョンが必要です。これを確認するには、Helmバージョンの互換性を範囲内の最高バージョンであるKubernetes v1.32と照らし合わせて確認してください。
* *結果:*Helm v3.17とHelm v3.18の両方がKubernetes v1.30-v1.32範囲をサポートしていることがわかります。
* どちらも動作しますが、サポートされているKubernetes範囲と重なる最新のHelmマイナーバージョンであるHelm v3.18を推奨します。

[#_additional_notes]
== 追加の注意事項

* Rancher v2.5をインストールまたはアップグレードするには、Helm v3.2.x以上が必要です。
* Helmコマンドを自動的に実行するツール（Terraformなど）を使用する場合は、正しいHelmバージョンを使用するように設定されていることを確認する必要があります。
