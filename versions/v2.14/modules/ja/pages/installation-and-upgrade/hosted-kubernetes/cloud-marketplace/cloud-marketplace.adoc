Rancherは、特定のクラウドプロバイダーでホストされているインストールのサポートを簡単に購入できるように、クラウドマーケットプレイスとの統合を提供します。さらに、この統合は、Rancherサポートに提供できるsupportconfigバンドルを生成する機能も備えています。

この統合は AWS のみをサポートしています。
