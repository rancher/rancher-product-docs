[#_1_uninstall_the_adapter_chart_using_helm]
== 1.Helmを使用してアダプターのチャートをアンインストールします。

[,bash]
----
helm uninstall rancher-csp-adapter -n cattle-csp-adapter-system
----

[#_2_remove_the_namespace_created_for_the_adapter]
== 2.アダプター用に作成されたネームスペースを削除します。

[,bash]
----
kubectl delete ns cattle-csp-adapter-system
----

[#_3_optional_remove_any_outstanding_user_notifications]
== 3.（オプション）未処理のユーザー通知を削除します。

[,bash]
----
kubectl delete RancherUserNotification csp-compliance
----
