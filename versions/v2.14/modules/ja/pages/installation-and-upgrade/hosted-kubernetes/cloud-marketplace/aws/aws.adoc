[#_overview]
== 概要

Rancherは、ユーザーがSUSEとのサポート契約を購入できるAWSマーケットプレイスとの統合を提供しています。この統合により、より多くのクラスターをサポートし始める際に、サポートニーズを簡単に調整できます。

[#_limitations]
== 制限

* Rancher v2.6.7以上を実行している必要があります。
* Rancherは、追加のメトリクスが有効になっている状態でデプロイされる必要があります。
* RancherはEKSクラスターにインストールされる必要があります。
* AWSマーケットプレイスを通じてRancherサポートの権利を少なくとも1つ購入する必要があります。
* プロキシ/エアギャップのユースケースをサポートするために、追加のセットアップが必要な場合があります。詳細については、{xref-aws-cloud-marketplace-adapter-reqs}[前提条件]を参照してください。

[#_how_to_use]
== 使用方法

. {xref-aws-cloud-marketplace-adapter-reqs}[前提条件の手順]を完了してください。
. {xref-aws-cloud-marketplace-install-adapter}[CSPアダプターをインストール]してください。

[#_faq]
== FAQ（よくある質問と答え）

*後でより多くのノードのサポートを購入できますか？*

はい。最初にサポートを購入したAWSマーケットプレイスのエントリに移動し、権利の数を増やしてください。

*同じAWSアカウントで複数のRancherインスタンスを使用できますか？*

はい。ただし、Rancherがインストールされている各クラスターは、前提条件を遵守する必要があります。

さらに、特定の権利は同時に1つのRancher管理サーバーでのみ使用できることに注意してください。
