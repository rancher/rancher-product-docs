多くの企業環境では、オンプレミスで稼働しているサーバーやVMは直接インターネットにアクセスできず、セキュリティ上の理由からHTTP(S)プロキシを介して外部サービスに接続する必要があります。このチュートリアルでは、そのような環境で高可用性のRancherのインストールを設定する方法をステップバイステップで示します。

また、インターネットアクセスなしで完全にエアギャップされたRancherを設定することも可能です。この手順は{xref-air-gap}[Rancherドキュメント]に詳細に説明されています。
