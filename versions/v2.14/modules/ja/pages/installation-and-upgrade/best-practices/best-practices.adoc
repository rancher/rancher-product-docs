このガイドには、Rancherサーバーを運用するための推奨事項が含まれており、RancherがダウンストリームKubernetesクラスターを管理する状況で使用されることを意図しています。

[#_recommended_architecture_and_infrastructure]
== 推奨アーキテクチャとインフラストラクチャ

本{xref-tips-for-running-rancher}[ガイド]を参照して、Rancherサーバーを本番環境に設置するための一般的なアドバイスをご覧ください。

[#_deployment_strategies]
== 導入計画

この{xref-deployment-strategy}[ガイド]は、ダウンストリームKubernetesクラスターを管理するRancherサーバーに対して、地域デプロイメント戦略とハブ＆スポークデプロイメント戦略のどちらが適しているかを選択するのに役立つように設計されています。

[#_installing_rancher_in_a_vmware_vsphere_environment]
== VMware vSphere環境にRancherをインストールする

この{xref-rancher-on-vsphere}[ガイド]は、VMwareによって文書化された標準のvSphereベストプラクティスに加えて、vSphere環境にRancherをインストールするための参照アーキテクチャを概説しています。
