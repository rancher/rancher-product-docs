[CAUTION]
====

これらのガイドの目的は、Rancher を評価するために使用できるサンドボックスを迅速に立ち上げることです。これらのガイドは、プロダクション環境を対象としていません。包括的なセットアップ手順については、{xref-install-upgrade}[インストール]を参照してください。
====


このドキュメントのセクションを使用して、Rancher 2.x のデプロイメントとテストを迅速に開始してください。簡単な Rancher セットアップといくつかの一般的なユースケースに関する指示が含まれています。将来的にこのセクションにさらにコンテンツを追加する予定です。

以下のためのクイックスタートガイドがあります：

* {xref-qs-deploy-rancher}[Rancher サーバーのデプロイメント]：ご都合の良い方法で、Rancher の実行を開始してください。
* {xref-qs-deploy-workloads}[ワークロードのデプロイメント]：シンプルな https://kubernetes.io/docs/concepts/workloads/[ワークロード] をデプロイして、それを公開し、クラスターの外部からアクセスできるようにします。
