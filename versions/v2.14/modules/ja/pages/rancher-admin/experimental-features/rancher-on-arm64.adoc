[CAUTION]
====
ARM64混合アーキテクチャプラットフォームでの実行は現在実験的な機能であり、Rancherではまだ正式にサポートされていません。したがって、プロダクション環境でARM64混合アーキテクチャに基づくノードの使用は推奨しません。
====

ARM64プラットフォームを使用する際に利用可能なオプションは次のとおりです：

* カスタムクラスターを作成し、ARM64ベースのノードを追加する
 ** Kubernetesクラスターのバージョンは1.12以上である必要があります
* ARM64ベースのノードを含むクラスターのインポート
 ** Kubernetesクラスターのバージョンは1.12以上である必要があります

クラスターのプロビジョニングに応じて、{xref-rke2-config}[RKE2クラスター設定オプション]または{xref-k3s-config}[K3sクラスター設定オプション]を参照して、詳細情報を確認してください。

次の機能はテストされていません：

* 監視、アラート、通知、パイプライン、ログ記録
* カタログからアプリを起動する
