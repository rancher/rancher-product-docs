[CAUTION]
====
RoleTemplate 集約は、RoleTemplates、ClusterRoleTemplateBindings、および ProjectRoleTemplateBindings に使用される RBAC アーキテクチャを変更する v2.13 の実験的な機能です。**本番環境ではサポートされていません**。v2.13 と v2.14 の間で破壊的な変更が発生する可能性があります。
====

RoleTemplate 集約は、Kubernetes 機能 https://kubernetes.io/docs/reference/access-authn-authz/rbac/#aggregated-clusterroles[集約された ClusterRoles] を使用して、RoleTemplates、ClusterRoleTemplateBindings、および ProjectRoleTemplateBindings を実装します。新しいアーキテクチャは、Rancher クラスターおよびダウンストリーム クラスターの両方で、RBAC オブジェクト（Roles、RoleBindings、ClusterRoles、および ClusterRoleBindings）の純減をもたらします。

この機能がスケーラビリティとパフォーマンスを向上させる方法についての詳細は、 https://www.suse.com/c/rancher_blog/fewer-bindings-more-power-ranchers-rbac-boost-for-enhanced-performance-and-scalability/[Rancher ブログ投稿]をご覧ください。

|===
| 環境変数キー | デフォルト値 | 説明

| `aggregated-roletemplates`
| `false`
| [ベータ] RoleTemplates が生成された RBAC ロールのために集約を使用するようにします。
|===

この機能フラグの値はインストール時にロックされ、UI では機能フラグの横にロックシンボルが表示されます。つまり、この機能は Rancher の初回インストール時にのみ設定できます。その後、値を変更しようとすると拒否されます。
