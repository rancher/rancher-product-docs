{xref-fleet}[Fleetによる継続的デリバリ]はRancherにプリインストールされており、完全に無効にすることはできません。ただし、GitOps継続的デリバリのためのFleet機能は、`continuous-delivery`機能フラグを使用して無効にすることができます。

この機能を有効または無効にするには、{xref-experimental-features}[実験的機能を有効にするためのメインページに関する指示]を参照してください。

|===
| 環境変数キー | デフォルト値 | 説明

| `continuous-delivery`
| `true`
| このフラグは、FleetのGitOps継続的デリバリ機能を無効にします。
|===

Rancher v2.5.xでFleetが無効になっていた場合、Rancherがv2.6.xにアップグレードされると有効になります。Fleetの継続的デリバリ部分のみが無効にできます。`continuous-delivery`が無効になると、`gitjob`デプロイメントはRancherサーバーのローカルクラスタにデプロイされなくなり、`continuous-delivery`はRancher UIに表示されません。
