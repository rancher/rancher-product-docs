Rancherの体験をパーソナライズするために設定を行うことができます。設定を変更するには:

. 右上隅のユーザアバターをクリックします。
. *Preferences*をクリックします。

[#_language]
== 言語

Rancher UIの言語を選択します。指定できる値は、次のとおりです。

* 英語
* 簡体字中国語 (Simplified Chinese)

[#_theme]
== テーマ

Rancher UIの背景色を選択します。*自動*を選択すると、背景色は午後6時に明るい色から暗い色に変わり、午前6時に再び明るい色に戻ります。

[#_login_landing_page]
== ログインランディングページ

ログイン後に表示されるページを選択します。指定できる値は、次のとおりです。

* ホームページ。
* 最後に訪れたエリア。
* 選択した特定のクラスター。

[#_display_settings]
== 表示設定

特定の情報の表示方法を選択します:

* 日付形式
* 時刻形式
* ページあたりのテーブル行数
* サイドメニューに表示するクラスターの数

[#_confirmation_setting]
== 確認設定

ノードプールを縮小する際に確認を求めるかどうかを選択します。

[#_advanced_features]
== 拡張機能

* 「APIで表示」を有効にします。
* Rancherによって管理されているシステムのネームスペースを表示します（編集や削除は意図していません）。
* ダーク/ライトテーマのキーボードショートカット切り替えを有効にします（shift+T）。
* すべてのタイプの説明を非表示にします。
* 拡張機能の開発者機能を有効にします。

[#_yaml_editor_key_mapping]
== YAMLエディタのキー割り当て

* 通常の人間（Sublime）
* Emacs
* Vim

[#_helm_charts]
== Helmチャート

リリースされたHelmチャートのみを表示するか、プレリリースバージョンも含めるかを選択します。バージョンは、 https://semver.org/#spec-item-9[仕様]が https://semver.org/[セマンティックバージョニング2.0.0]によって定義されている場合、プレリリースと見なされます。例えば、バージョンが`0.1.3-dev.12ab4f`のHelmチャートは、表示されるために`Include Prerelease Versions`を選択する必要があります。
