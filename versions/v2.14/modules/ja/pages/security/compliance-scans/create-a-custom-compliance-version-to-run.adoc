一部の Kubernetes クラスターのセットアップでは、コンプライアンス テストのカスタム設定が必要な場合があります。たとえば、Kubernetes の設定ファイルや証明書へのパスが、アップストリームのコンプライアンスがそれらを探す標準の場所とは異なる場合があります。

`rancher-compliance`アプリケーションを使用して、クラスター スキャンを実行するためのカスタムコンプライアンス バージョンを作成することが可能になりました。

詳細については、{xref-compliance-scan-custom-benchmark}[このページ]を参照してください。
