スケジュールに従って実行されるスキャンのために、アラートを送信するように設定できます。

[NOTE]
.前提条件:
====

`rancher-compliance`のアラートを有効にする前に、`rancher-monitoring`アプリケーションをインストールし、受信者とルートを設定してください。詳細については、{xref-receivers}[このセクション]を参照してください。

`rancher-compliance`アラートのルートを設定する際に、キーと値のペア`job: rancher-compliance-scan`を使用して一致を指定できます。An example route configuration is {xref-receivers}#_example_route_config_for_compliance_scan_alerts[here.]
====


`rancher-compliance` Helmチャートをインストールまたはアップグレードする際に、`true`で次のフラグを`values.yaml`に設定してください。

[,yaml]
----
alerts:
  enabled: true
----
