`k3s-selinux`はK3sのポリシーを提供します。K3sインストーラスクリプトがRPMベースのディストリビューションで実行されていることを検出すると、自動的にインストールされます。

`k3s-selinux`のGitHubリポジトリは https://github.com/rancher/k3s-selinux[こちら]です。

SELinux対応のホストにK3sをインストールする方法についての詳細は、 https://documentation.suse.com/cloudnative/k3s/latest/en/advanced.html#_selinux_support[K3sのドキュメント]を参照してください。
