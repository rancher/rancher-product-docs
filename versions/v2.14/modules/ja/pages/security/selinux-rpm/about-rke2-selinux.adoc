`rke2-selinux`はRKE2のポリシーを提供します。RKE2インストーラスクリプトがRPMベースのディストリビューションで実行されていることを検出すると、自動的にインストールされます。

`rke2-selinux`のGitHubリポジトリは https://github.com/rancher/rke2-selinux[こちらです。]

SELinux対応のホストにRKE2をインストールする方法についての詳細は、 https://documentation.suse.com/cloudnative/rke2/latest/en/install/methods.html#rpm[RKE2のドキュメントをご覧ください。]
