https://en.wikipedia.org/wiki/Security-Enhanced_Linux[セキュリティ強化Linux（SELinux）]はLinuxへのセキュリティ強化です。

Red Hatによって開発され、Linux上での必須アクセス制御（MAC）の実装です。必須アクセス制御により、システムの管理者は、アプリケーションやユーザーがファイル、デバイス、ネットワーク、プロセス間通信などの異なるリソースにアクセスする方法を定義できます。SELinuxは、OSをデフォルトで制限的にすることでセキュリティを強化します。

歴史的に政府機関によって使用されてきたSELinuxは、現在では業界標準となり、SUSE Linux Enterprise 16やopenSUSE TumbleweedなどのSUSEディストリビューション、Rocky LinuxなどのRHELベースのOSでデフォルトで有効になっています。SELinuxがシステムで有効かつ強制されているかを確認するには、`getenforce`を使用してください。

----
# getenforce
Enforcing
----

SELinuxが強制されているホスト上でRancher製品が正常に機能するために、3つのRPM（RPMパッケージ）を提供しています。

* {xref-rancher-selinux}[`rancher-selinux`]
* {xref-rke2-selinux}[`rke2-selinux`]
* {xref-k3s-selinux}[`k3s-selinux`]
