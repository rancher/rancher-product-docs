アカウントアクセス情報はクラウド資格情報として保存されます。クラウド資格情報はKubernetesシークレットとして保存されます。複数のノードテンプレートが同じクラウド資格情報を使用できます。既存のクラウド資格情報を使用するか、新しいものを作成できます。

[#_droplet_options]
== ドロップレットオプション

*ドロップレットオプション*は、クラスターの地理的リージョンおよび仕様をプロビジョニングします。

[#_docker_daemon]
== Docker デーモン

Dockerを使用する場合、 https://docs.docker.com/engine/docker-overview/#the-docker-daemon[Docker デーモン]の設定オプションには以下が含まれます：

* *Labels:*ラベルに関する情報は、 https://docs.docker.com/config/labels-custom-metadata/[Dockerオブジェクトラベルのドキュメント]を参照してください。
* *Docker Engine インストール URL:*インスタンスにインストールされるDockerのバージョンを決定します。
* *レジストリミラー:*Docker デーモンによって使用される Docker レジストリミラー
* *その他の高度なオプション:*https://docs.docker.com/engine/reference/commandline/dockerd/[Docker デーモンオプションリファレンス]を参照してください。
