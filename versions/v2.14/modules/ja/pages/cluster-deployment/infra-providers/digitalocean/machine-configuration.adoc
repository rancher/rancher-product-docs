DigitalOcean や Droplets に関する詳細は、 https://docs.digitalocean.com/products/compute/[公式ドキュメント]を参照してください。

[#_region]
== 地域

Droplets が作成される https://docs.digitalocean.com/glossary/region/[リージョン]を設定します。

[#_size]
== サイズ

Droplets の https://docs.digitalocean.com/products/droplets/resources/choose-plan/[サイズ]を設定します。

[#_os_image]
== OS イメージ

Droplets が作成されるオペレーティングシステム https://docs.digitalocean.com/products/images/[イメージ]を設定します。

[#_monitoring]
== 監視

追加の https://docs.digitalocean.com/products/monitoring/[監視]のために DigitalOcean エージェントを有効にします。

[#_ipv6]
== IPv6

Droplets に対して IPv6 を有効にします。

詳細については、 https://docs.digitalocean.com/products/networking/ipv6[DigitalOcean IPv6 ドキュメント]を参照してください。

[#_private_networking]
== プライベートネットワーキング

Droplets に対してプライベートネットワーキングを有効にします。

[#_droplet_tags]
== Droplet タグ

Droplet にタグ（ラベル）を適用します。タグには文字、数字、コロン、ダッシュ、アンダースコアのみを含めることができます。たとえば、`my_server`と指定します。
