インツリーVMware vSphereクラウドプロバイダーをセットアップするには、RancherでvSphereクラスターを作成する際に、以下の手順に従ってください。

. *クラウドプロバイダー*オプションを`Custom`または`Custom (In-Tree)`に設定します。
. *Edit as YAML*をクリックします。
. 事前に入力されたクラスターYAMLに以下の構造を挿入します。この構造は`rancher_kubernetes_engine_config`の下に配置する必要があります。`name`が_必ず_ `vsphere`に設定されていることに注意してください。
+
[,yaml]
----
 rancher_kubernetes_engine_config:
   cloud_provider:
       name: vsphere
       vsphereCloudProvider:
           [Insert provider configuration]
----

RancherはKubernetesクラスターをプロビジョニングするためにRKE（Rancher Kubernetes Engine）を使用します。 https://rancher.com/docs/rke/latest/en/config-options/cloud-providers/vsphere/config-reference/[RKEドキュメントのvSphere構成リファレンス]を参照して、`vsphereCloudProvider`ディレクティブのプロパティの詳細を確認してください。
