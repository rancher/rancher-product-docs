ノードにAzure VMを使用している場合、クラスターのストレージクラスとして https://docs.microsoft.com/en-us/azure/aks/azure-files-dynamic-pv[Azure files]を使用できます。

Azureプラットフォームに必要なストレージリソースを作成させるために、以下の手順に従ってください：

. {xref-azure-cloud-provider}[Azureクラウドプロバイダーを設定します。]
. クラスターに接続するために`kubectl`を設定します。
. サービスアカウントのために`ClusterRole`と`ClusterRoleBinding`のマニフェストをコピーします： +
  ```yml +
  -- +
  apiVersion: rbac.authorization.k8s.io/v1 +
  kind:ClusterRole +
  metadata: +
    name: system:azure-cloud-provider +
  rules:
 ** apiGroups: [''] +
resources: ['secrets'] +
verbs:     ['get','create'] +
  -- +
  apiVersion: rbac.authorization.k8s.io/v1 +
  kind:ClusterRoleBinding
  metadata:
name: system:azure-cloud-provider
  roleRef:
kind:ClusterRole
apiGroup: rbac.authorization.k8s.io
name: system:azure-cloud-provider
  subjects:
 ** kind:ServiceAccount +
name: persistent-volume-binder +
namespace: kube-system +
  ```
. これらをクラスター内で次のいずれかのコマンドで作成します。
+
----
# kubectl create -f <MANIFEST>
----
