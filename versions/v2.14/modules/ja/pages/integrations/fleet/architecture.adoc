Fleetは、Gitから提供される生のKubernetes YAML、Helmチャート、Kustomize、またはこれら3つの任意の組み合わせによるデプロイメントを管理することができます。ソースに関係なく、すべてのリソースは動的にHelmチャートに変換され、Helmがクラスタ内のすべてのリソースをデプロイするためのエンジンとして使用されます。これにより、高い制御性、一貫性、監査可能性が得られます。Fleetは、単にスケールできるだけでなく、クラスタに何が正確にインストールされているかを高いレベルで管理し把握できる点にも注力しています。
