Rancher Desktopは、デスクトップからクラウドネイティブアプリケーションを開発およびテストするための必須ツールをまとめています。

クラウド環境向けのアプリをローカルマシンで作業している場合、通常は多くの準備が必要です。コンテナランタイムを選択し、Kubernetesや一般的なユーティリティをインストールし、場合によっては仮想マシンを設定する必要があります。コンポーネントを個別にインストールし、それらを連携させるのは時間がかかるプロセスです。

複雑さを軽減するために、Rancher Desktopはチームに以下の主要機能を提供します：

* macOS、Linux、Windowsオペレーティングシステムでのシンプルで簡単なインストール。
* すぐに使用できる軽量のKubernetesディストリビューションであるK3s。
* Kubernetesのバージョンを簡単に切り替える機能。
* Rancherが提供する、ローカルクラスタを探索するためのGUIベースのクラスターダッシュボード。
* コンテナエンジンを選択する自由：dockerd（moby）またはcontainerd。
* アプリケーションをニーズに合わせて設定するための設定オプション。
* コンテナ、Kubernetesベースの開発、および運用ワークフローに必要なツールをバンドル。
* バンドルされたツールを最新の状態に保つための定期的な更新。
* VS CodeやSkaffoldを含む人気のあるツール/IDEとの統合。
* イメージとレジストリのアクセス制御。
* Docker拡張機能のサポート。

https://rancherdesktop.io[Rancher Desktop]のウェブサイトを訪れ、 https://docs.rancherdesktop.io/[ドキュメント]を読んで詳細を学んでください。

マシンにRancher Desktopをインストールするには、 https://docs.rancherdesktop.io/getting-started/installation[インストールガイド]を参照してください。

[#_trying_rancher_on_rancher_desktop]
== Rancher DesktopでRancherを試す

Rancher Desktopは、コンテナ化されたHelmベースのアプリケーションを簡単に試すために必要なセットアップとツールを提供します。Rancher Desktopを使用してRancher Kubernetes Managementプラットフォームを始めるには、この https://docs.rancherdesktop.io/how-to-guides/rancher-on-rancher-desktop[ハウツー]に従ってください。
