Longhornは公式の https://cncf.io/[Cloud Native Computing Foundation（CNCF）プロジェクト]であり、どこでも実行できるKubernetes用の強力なクラウドネイティブ分散ストレージプラットフォームを提供します。Rancherと組み合わせることで、LonghornはKubernetes環境における高可用性の永続ブロックストレージのデプロイメントを簡単、迅速、かつ信頼性の高いものにします。

[#__longhorn_product_name_with_rancher]
== {longhorn-product-name} とRancher

Rancher PrimeとLonghornを使用することで、ユーザーはRancherカタログを介してワンクリックで簡単にデプロイでき、管理されたクラスターのライフサイクル管理を行うことができます。これにより、ユーザーはインストールやアップグレードを行い、円滑な運用のためのドレイン操作も実施することができます。RancherとLonghornを組み合わせることで、Windowsとの混合クラスターサポート、Rancherホストイメージ、Rancherを通じたUIプロキシアクセス、LonghornメトリクスによるRancherモニタリングも提供されます。
