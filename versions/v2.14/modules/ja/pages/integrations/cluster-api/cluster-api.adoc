https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}] は、Cluster API (CAPI) と Rancher を統合し、プロビジョニングされた Kubernetes クラスターのライフサイクルを管理する https://kubernetes.io/docs/concepts/extend-kubernetes/operator/#operators-in-kubernetes[Kubernetes Operator] です。Rancher Turtles を使用すると、次のことができます：

* CAPI でプロビジョニングされたクラスターに Rancher Cluster Agent をインストールして、CAPI クラスターを Rancher にインポートできます。
* https://cluster-api-operator.sigs.k8s.io/[CAPI Operator] を {url-turtles-capiprovider}[CAPIProvider] リソースで管理します。

{xref-turtles-overview}[Overview] セクションでは、インストールオプション、Rancher Turtles のアーキテクチャ、および簡単なデモについて説明します。詳細については、 https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name} ドキュメント] を参照してください。
