Para mais detalhes sobre a DigitalOcean, Droplets, consulte a https://docs.digitalocean.com/products/compute/[documentação oficial].

[#_region]
== Região

Configure a https://docs.digitalocean.com/glossary/region/[região] onde os Droplets são criados.

[#_size]
== Tamanho

Configure o https://docs.digitalocean.com/products/droplets/resources/choose-plan/[tamanho] dos Droplets.

[#_os_image]
== Imagem do SO

Configure a https://docs.digitalocean.com/products/images/[imagem] do sistema operacional a partir da qual os Droplets são criados.

[#_monitoring]
== Monitoramento

Ative o agente DigitalOcean para https://docs.digitalocean.com/products/monitoring/[monitoramento] adicional.

[#_ipv6]
== IPv6

Ative o IPv6 para os Droplets.

Para mais informações, consulte a https://docs.digitalocean.com/products/networking/ipv6[documentação do IPv6 da DigitalOcean].

[#_private_networking]
== Rede Privada

Ative a rede privada para os Droplets.

[#_droplet_tags]
== Tags de Droplet

Aplique uma tag (rótulo) a um Droplet. As tags podem conter apenas letras, números, dois-pontos, traços e sublinhados. Por exemplo, `my_server`.
