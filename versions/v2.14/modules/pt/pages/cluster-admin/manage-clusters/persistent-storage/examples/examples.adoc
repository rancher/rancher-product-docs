O Rancher suporta armazenamento persistente com uma variedade de plugins de volume. No entanto, antes de usar qualquer um desses plugins para vincular armazenamento persistente às suas cargas de trabalho, você precisa configurar o armazenamento em si, seja uma solução baseada em nuvem de um provedor de serviços ou uma solução local que você gerencia.

Para sua conveniência, o Rancher oferece documentação sobre como configurar alguns dos métodos de armazenamento populares:
