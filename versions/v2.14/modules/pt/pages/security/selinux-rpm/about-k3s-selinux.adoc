`k3s-selinux` fornece políticas para o K3s. Ele é instalado automaticamente quando o script de instalação do K3s detecta que está sendo executado em uma distribuição baseada em RPM.

O repositório do GitHub `k3s-selinux` está https://github.com/rancher/k3s-selinux[aqui].

Para mais informações sobre como instalar o K3s em hosts com SELinux habilitado, consulte a https://documentation.suse.com/cloudnative/k3s/latest/en/advanced.html#_selinux_support[documentação do K3s].
