https://en.wikipedia.org/wiki/Security-Enhanced_Linux[Linux com Segurança Aprimorada (SELinux)] é uma melhoria de segurança para o Linux.

Desenvolvido pela Red Hat, é uma implementação de controles de acesso obrigatórios (MAC) no Linux. Os controles de acesso obrigatórios permitem que um administrador de sistema defina como aplicativos e usuários podem acessar diferentes recursos, como arquivos, dispositivos, redes e comunicação entre processos. O SELinux também aprimora a segurança tornando um sistema operacional restritivo por padrão.

Após ser historicamente utilizado por agências governamentais, o SELinux agora é um padrão da indústria e está habilitado por padrão em distribuições SUSE, como SUSE Linux Enterprise 16, distribuições openSUSE, como openSUSE Tumbleweed, e sistemas operacionais baseados em RHEL, como Rocky Linux. Para verificar se o SELinux está habilitado e em vigor em seu sistema, use `getenforce`:

----
# getenforce
Enforcing
----
