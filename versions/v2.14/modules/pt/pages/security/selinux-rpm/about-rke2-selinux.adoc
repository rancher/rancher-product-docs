`rke2-selinux` fornece políticas para o RKE2. Ele é instalado automaticamente quando o script de instalação do RKE2 detecta que está sendo executado em uma distribuição baseada em RPM.

O `rke2-selinux` repositório do GitHub está https://github.com/rancher/rke2-selinux[aqui.]

Para mais informações sobre como instalar o RKE2 em hosts com SELinux habilitado, consulte a https://documentation.suse.com/cloudnative/rke2/latest/en/install/methods.html#rpm[documentação do RKE2.]
