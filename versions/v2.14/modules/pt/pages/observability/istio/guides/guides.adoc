[WARNING]
====
https://github.com/rancher/charts/tree/release-v2.11/charts/rancher-istio[Rancher-Istio] foi descontinuado desde o Rancher v2.12.0; recorra à https://apps.rancher.io[SUSE Application Collection] versão do Istio para segurança aprimorada (incluída nas assinaturas do SUSE Rancher Prime).
Informações detalhadas podem ser encontradas em https://forums.suse.com/t/deprecation-of-rancher-istio/45043[este anúncio].
====

Esta seção descreve como ativar o Istio e começar a usá-lo em seus projetos.

Se você usar o Istio para gestão de tráfego, precisará permitir tráfego externo para o cluster. Nesse caso, você precisará seguir todos os passos abaixo.

[#_prerequisites]
== Pré-requisitos

Este guia assume que você já {xref-install-upgrade}[instalou o Rancher,] e que você já {xref-cluster-deployment}[provisionou um cluster Kubernetes separado] no qual instalará o Istio.

Os nós em seu cluster devem atender aos {xref-istio-cpu-memory-allocations}[requisitos de CPU e memória.]

As cargas de trabalho e serviços que você deseja que sejam controlados pelo Istio devem atender aos https://istio.io/docs/setup/additional-setup/requirements/[requisitos do Istio.]

[#_install]
== Instale o

[TIP]
.Dica de Configuração Rápida:
====

Se você não precisar que o tráfego externo chegue ao Istio e apenas quiser configurar o Istio para monitorar e rastrear o tráfego dentro do cluster, pule os passos para {xref-istio-gateway-setup}[configurar o gateway do Istio] e {xref-istio-traffic-management-setup}[configurar os componentes do Istio para gestão de tráfego.]
====


. {xref-enable-istio-in-cluster}[Ativar o Istio no cluster.]
. {xref-enable-istio-in-namespace}[Ativar o Istio em todos os namespaces onde você deseja usá-lo.]
. {xref-istio-sidecar}[Adicione implantações e serviços que tenham o sidecar do Istio injetado.]
. {xref-istio-gateway-setup}[Configure o gateway do Istio.]
. {xref-istio-traffic-management-setup}[Configure os componentes do Istio para gestão de tráfego.]
. {xref-istio-generate-and-view-traffic}[Gere tráfego e veja o Istio em ação.]
