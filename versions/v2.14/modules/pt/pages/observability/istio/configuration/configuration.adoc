[#_egress_support]
== Suporte a Egress

Por padrão, o Egress gateway está desativado, mas pode ser habilitado na instalação ou ao fazer upgrade através do values.yaml ou via <<_overlay_file,arquivo de sobreposição>>.

[#_enabling_automatic_sidecar_injection]
== Habilitando a Injeção Automática de Sidecar

A injeção automática de sidecar está desativada por padrão. Para habilitar isso, defina o `sidecarInjectorWebhook.enableNamespacesByDefault=true` no values.yaml na instalação ou ao fazer upgrade. Isso habilita automaticamente a injeção de sidecar do Istio em todos os novos namespaces que são implantados.

[#_overlay_file]
== Arquivo de Sobreposição

Um Arquivo de Sobreposição é projetado para suportar uma configuração extensa da sua instalação do Istio. Ele permite que você faça alterações em quaisquer valores disponíveis na https://istio.io/latest/docs/reference/config/istio.operator.v1alpha1/[API do IstioOperator]. Isso garantirá que você possa personalizar a instalação padrão para se adequar a qualquer cenário.

O Arquivo de Sobreposição adicionará configuração sobre a instalação padrão fornecida pelo chart do Istio. Isso significa que você não precisa redefinir os componentes que já foram definidos para a instalação.

Para mais informações sobre Arquivos de Sobreposição, consulte a https://istio.io/latest/docs/setup/install/istioctl/#configure-component-settings[documentação do Istio.]

[#_selectors_and_scrape_configs]
== Seletores e Configurações de Scrape

O app de Monitoramento define `prometheus.prometheusSpec.ignoreNamespaceSelectors=false`, que habilita o monitoramento em todos os namespaces por padrão. Isso garante que você possa visualizar tráfego, métricas e gráficos para recursos implantados em um namespace com o rótulo `istio-injection=enabled`.

Se você deseja limitar o Prometheus a namespaces específicos, defina `prometheus.prometheusSpec.ignoreNamespaceSelectors=true`. Uma vez que você fizer isso, deve realizar algumas configurações adicionais para continuar monitorando seus recursos.

Para detalhes, consulte {xref-selectors-and-scrape-configurations}[esta seção.]
