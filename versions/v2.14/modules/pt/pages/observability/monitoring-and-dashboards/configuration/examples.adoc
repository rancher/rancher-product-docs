[#_prometheusrule]
== PrometheusRule

Uma PrometheusRule contém as regras de alerta e gravação que você normalmente colocaria em um https://prometheus.io/docs/prometheus/latest/configuration/recording_rules/[arquivo de regras do Prometheus].

Para uma abordagem mais detalhada, o campo `ruleSelector` em um recurso Prometheus pode selecionar quais PrometheusRules devem ser carregadas no Prometheus, com base nos rótulos anexados aos recursos PrometheusRules.

Consulte a https://prometheus-operator.dev/docs/developer/alerting/[documentação do Prometheus Operator] para um exemplo de PrometheusRule.
