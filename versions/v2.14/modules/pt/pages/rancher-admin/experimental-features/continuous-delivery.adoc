{xref-fleet}[A entrega contínua com Fleet] vem pré-instalada no Rancher e não pode ser totalmente desativada. No entanto, o recurso Fleet para entrega contínua GitOps pode ser desativado usando a feature flag `continuous-delivery`.

Para habilitar ou desabilitar este recurso, consulte as instruções na {xref-experimental-features}[página principal sobre como habilitar recursos experimentais.]

|===
| Chave da Variável de Ambiente | Valor Padrão | Descrição

| `continuous-delivery`
| `true`
| Esta feature flag desativa o recurso de entrega contínua GitOps do Fleet.
|===

Se o Fleet foi desativado no Rancher v2.5.x, ele será ativado se o Rancher fizer upgrade para v2.6.x. Apenas a parte de entrega contínua do Fleet pode ser desativada. Quando `continuous-delivery` é desativado, a implantação `gitjob` não é mais implantada no cluster local do servidor Rancher, e `continuous-delivery` não é exibido na interface do usuário do Rancher.
