Dentro do Rancher, cada usuário tem um número de configurações associadas ao seu login: preferências pessoais, chaves de API, etc. Você pode configurar essas configurações escolhendo no *menu Configurações do Usuário*. Você pode abrir este menu clicando no seu avatar, localizado dentro do menu principal.

image::user-settings.png[Menu de Configurações do Usuário]

As configurações de usuário disponíveis são:

* {xref-user-api-keys}[API & Chaves]: Se você deseja interagir com o Rancher programaticamente, precisa de uma chave de API. Siga as instruções nesta seção para obter uma chave.
* {xref-cloud-credentials}[Credenciais de Nuvem]: Gerencie as credenciais de nuvem usadas por pools de máquinas para {xref-launch-k8s-with-rancher}[provisionar nós para clusters].
* {xref-user-preferences}[Preferências]: Define preferências superficiais para a interface do Rancher.
* Efetuar Logout: Encerra sua sessão de usuário.
