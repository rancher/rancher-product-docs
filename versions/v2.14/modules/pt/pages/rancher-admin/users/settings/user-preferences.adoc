Você pode definir preferências para personalizar sua experiência no Rancher. Para alterar as configurações de preferência:

. Clique no seu avatar de usuário no canto superior direito.
. Clique em *Preferências*.

[#_language]
== Idioma

Escolha seu idioma para a interface do Rancher. Estas são as opções:

* Inglês
* Chinês Simplificado (简体中文)

[#_theme]
== Tema

Escolha a cor de fundo para a interface do Rancher. Se você escolher *Auto*, a cor de fundo muda de clara para escura às 18h e depois volta a mudar às 6h.

[#_login_landing_page]
== Página inicial de login

Escolha a página exibida após o login. Estas são as opções:

* A página inicial.
* A área que você visitou por último.
* Um cluster específico de sua escolha.

[#_display_settings]
== Configuração de tela

Escolha como certas informações são exibidas:

* Formato de data
* Formato de horário
* Linhas da tabela por página
* Número de clusters a serem exibidos no menu lateral

[#_confirmation_setting]
== Configuração de Confirmação

Escolha se deseja pedir confirmação ao reduzir pools de nós.

[#_advanced_features]
== Recursos avançados

* Ativar "Ver na API".
* Mostrar namespaces do sistema gerenciados pelo Rancher (não destinados à edição ou exclusão).
* Ativar atalho de teclado para alternar entre o tema escuro/claro (shift+T).
* Ocultar todas as descrições de tipo.
* Ativar recursos de desenvolvedor de extensão.

[#_helm_charts]
== Gráficos do Helm

Escolha se deseja exibir apenas os Helm charts lançados ou incluir também versões de pré-lançamento. Uma versão é considerada um pré-lançamento se estiver em conformidade com https://semver.org/#spec-item-9[a especificação] definida por https://semver.org/[Versionamento Semântico 2.0.0]. Por exemplo, um Helm chart com uma versão de `0.1.3-dev.12ab4f` requer que `Include Prerelease Versions` seja selecionado para ser exibido.
