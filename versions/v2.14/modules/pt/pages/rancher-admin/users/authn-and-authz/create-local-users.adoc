A autenticação local é o padrão até que você configure um provedor de autenticação externo. O Rancher armazena informações da conta do usuário, como nomes de usuário e senhas, localmente. Por padrão, o usuário `admin` que faz login no Rancher pela primeira vez é um usuário local.

[#_adding_local_users]
== Adicionando Usuários Locais

Independentemente de você usar autenticação externa, deve criar alguns usuários de autenticação local para que possa continuar usando o Rancher se o seu serviço de autenticação externa encontrar problemas.

. No canto superior esquerdo, clique em *☰ > Usuários e Autenticação*.
. No menu de navegação à esquerda, clique em *Usuários*.
. Clique em *Criar*.
. Preencha o formulário *Adicionar Usuário*.
. Clique em *Criar*.
