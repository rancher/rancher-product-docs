Este guia contém nossas recomendações para executar o servidor Rancher e destina-se a ser usado em situações nas quais o Rancher gerencia clusters Kubernetes downstream.

[#_recommended_architecture_and_infrastructure]
== Arquitetura e Infraestrutura Recomendadas

Consulte este {xref-tips-for-running-rancher}[guia] para nossas orientações gerais sobre como configurar o servidor Rancher para uma instalação em produção.

[#_deployment_strategies]
== Estratégias de implantação

Este {xref-deployment-strategy}[guia] foi projetado para ajudá-lo a escolher se uma estratégia de implantação regional ou uma estratégia de implantação hub-and-spoke é melhor para um servidor Rancher que gerencia clusters Kubernetes downstream.

[#_installing_rancher_in_a_vmware_vsphere_environment]
== Instalando o Rancher em um Ambiente VMware vSphere

Este {xref-rancher-on-vsphere}[guia] descreve uma arquitetura de referência para instalar o Rancher em um ambiente vSphere, além das melhores práticas padrão do vSphere conforme documentado pela VMware.
