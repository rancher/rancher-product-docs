Em muitos ambientes empresariais, servidores ou VMs em execução local não têm acesso direto à Internet, mas devem se conectar a serviços externos através de um proxy HTTP(S) por razões de segurança. Este tutorial mostra passo a passo como configurar uma instalação do Rancher altamente disponível em tal ambiente.

Alternativamente, também é possível configurar o Rancher completamente air-gapped, sem acesso à Internet. Este processo é descrito em detalhes no {xref-air-gap}[Rancher docs].
