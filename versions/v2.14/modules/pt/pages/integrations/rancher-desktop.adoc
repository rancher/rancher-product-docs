O Rancher Desktop reúne ferramentas essenciais para desenvolver e testar aplicativos nativos de nuvem a partir do seu desktop.

Se você está trabalhando em sua máquina local em apps destinados a ambientes de nuvem, normalmente precisa de muita preparação. Você precisa selecionar um tempo de execução de contêiner, instalar o Kubernetes e utilitários populares, e possivelmente configurar uma máquina virtual. Instalar componentes individualmente e fazê-los funcionar juntos pode ser um processo demorado.

Para reduzir a complexidade, o Rancher Desktop oferece às equipes os seguintes recursos principais:

* Instalação simples e fácil nos sistemas operacionais macOS, Linux e Windows.
* K3s, uma distribuição Kubernetes leve e pronta para uso.
* A capacidade de alternar facilmente entre versões do Kubernetes.
* Um painel de controle de cluster baseado em GUI, alimentado pelo Rancher, para explorar seu cluster local.
* Liberdade para escolher seu mecanismo de contêiner: dockerd (moby) ou containerd.
* Configurações de preferência para configurar o aplicativo de acordo com suas necessidades.
* Ferramentas agrupadas necessárias para seu contêiner, para desenvolvimento baseado em Kubernetes e para fluxos de trabalho de operação.
* Atualizações periódicas para manter as ferramentas agrupadas atualizadas.
* Integração com ferramentas/IDEs populares, incluindo VS Code e Skaffold.
* Controle de acesso a imagens e registro.
* Suporte para extensões do Docker.

Visite o https://rancherdesktop.io[site do Rancher Desktop] e leia os https://docs.rancherdesktop.io/[docs] para saber mais.

Para instalar o Rancher Desktop em sua máquina, consulte o https://docs.rancherdesktop.io/getting-started/installation[guia de instalação].

== Experimentando o Rancher no Rancher Desktop

O Rancher Desktop oferece a configuração e as ferramentas necessárias para experimentar facilmente aplicativos conteinerizados e baseados em Helm. Você pode começar a usar a plataforma de gerenciamento Kubernetes do Rancher com o Rancher Desktop, seguindo este https://docs.rancherdesktop.io/how-to-guides/rancher-on-rancher-desktop[guia de procedimentos].
