https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}] é um https://kubernetes.io/docs/concepts/extend-kubernetes/operator/#operators-in-kubernetes[Operador Kubernetes] que gerencia o ciclo de vida de clusters Kubernetes provisionados, fornecendo integração entre sua API de Cluster (CAPI) e o Rancher. Com o Rancher Turtles, você pode:

* Importar clusters CAPI para o Rancher, instalando o Agente de Cluster do Rancher em clusters provisionados pelo CAPI.
* Gerenciar o https://cluster-api-operator.sigs.k8s.io/[Operador CAPI] usando o recurso {url-turtles-capiprovider}[CAPIProvider].

A seção {xref-turtles-overview}[Visão Geral] descreve opções de instalação, arquitetura do Rancher Turtles e uma breve demonstração. Para mais detalhes, veja a https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name} documentação].
