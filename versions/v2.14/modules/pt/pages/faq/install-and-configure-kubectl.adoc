`kubectl` é uma CLI para executar comandos em clusters Kubernetes. É necessário para muitas tarefas de manutenção e administrativas no Rancher 2.x.

[#_installation]
== Instalação

Veja https://kubernetes.io/docs/tasks/tools/install-kubectl/[Instalação do kubectl] para a instalação no seu sistema operacional.

[#_configuration]
== Configuração

Quando você cria um cluster Kubernetes com RKE2/K3s, o arquivo Kubeconfig é armazenado em `/etc/rancher/rke2/rke2.yaml` ou `/etc/rancher/k3s/k3s.yaml`, dependendo da distribuição escolhida. Esses arquivos são usados para configurar o acesso ao cluster Kubernetes.

Teste sua conectividade com `kubectl` e veja se você consegue obter a lista de nós.

----
kubectl get nodes
