Longhorn 是 https://cncf.io/[云原生计算基金会（CNCF）] 的官方项目，为 Kubernetes 提供强大的云原生分布式存储平台，可在任何环境中运行。与 Rancher 结合使用时，Longhorn 使在 Kubernetes 环境中部署高可用的持久块存储变得简单、快速且可靠。

[#__longhorn_product_name_with_rancher]
== {longhorn-product-name} 与 Rancher

通过 Rancher Prime 和 Longhorn，用户可以通过 Rancher 目录轻松实现一键部署，并对托管集群进行生命周期管理，从而能够进行安装、升级，并在平滑停机时执行排空操作。Longhorn 与 Rancher 还支持 Windows 混合集群、Rancher 托管镜像、通过 Rancher 进行 UI 代理访问，以及通过 Rancher 监控 Longhorn 指标。
