Fleet 可以管理来自 git 的原始 Kubernetes YAML、Helm 图表或 Kustomize，或三者任意组合的部署。无论来源如何，所有资源都会动态转换为 Helm 图表，Helm 被用作在集群中部署所有内容的引擎。这为您提供了高度的控制、一致性和可审计性。Fleet 不仅关注扩展能力，更致力于让用户对集群中安装的内容拥有高度的控制和可见性。
