在 Rancher v2.6.1 中引入的 Harvester 是一个基于 Kubernetes 的开源超融合基础设施 (HCI) 软件。Harvester 安装在裸机服务器上，提供集成虚拟化和分布式存储功能。尽管 Harvester 使用 Kubernetes 进行操作，但它不需要了解 Kubernetes 概念，使其更加用户友好。

[#__harvester_product_name_with_rancher]
== {harvester-product-name} 与 Rancher 一起使用

通过 Rancher Prime 和 Harvester，IT 操作员现在可以访问一个企业级、易于使用的基础设施平台，能够协调管理他们的虚拟机和 Kubernetes 集群。有关支持服务的更多信息，请参见 https://www.suse.com/suse-harvester/support-matrix/all-supported-versions/harvester-v1-2-0/[支持矩阵]。通过 Rancher 虚拟化管理功能，用户可以导入和管理多个 Harvester cluster。利用 Rancher 的身份验证功能和 RBAC 控制来支持多租户。
