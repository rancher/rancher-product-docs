SUSE Observability 是一个完整的可观测性解决方案，提供对集群和节点健康状况以及其上运行的工作负载的深入洞察。SUSE Observability 旨在为您提供对整个 Kubernetes 环境的清晰可见性，其全栈方法使您能够在单一平台上无缝探索从服务到基础设施的所有内容，消除了对多个可观测性工具的需求。

SUSE Observability 安全地收集和关联数据，提供对集群中现有和潜在问题的可操作洞察。这帮助您迅速解决当前问题，并采取预防措施应对未来挑战。

直观的仪表板突出问题区域并提供修复步骤，引导您从问题识别到根本原因分析，最终在最短的时间内解决问题。

有关更多信息以及如何在您的 SUSE Rancher 管理的 Kubernetes 集群中设置 SUSE Observability，请参考 https://documentation.suse.com/en-us/cloudnative/suse-observability/[文档]。
