https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}] 是一个 https://kubernetes.io/docs/concepts/extend-kubernetes/operator/#operators-in-kubernetes[Kubernetes Operator]，它通过提供您的集群 API (CAPI) 和 Rancher 之间的集成来管理已提供的 Kubernetes 集群的生命周期。使用 Rancher Turtles，您可以：

* 通过在 CAPI 提供的集群中安装 Rancher 集群代理，将 CAPI 集群导入 Rancher。
* 使用 https://cluster-api-operator.sigs.k8s.io/[CAPI Operator] 管理 {url-turtles-capiprovider}[CAPIProvider] 资源。

{xref-turtles-overview}[概述] 部分概述了安装选项、Rancher Turtles 架构和简要演示。有关更多详细信息，请参见 https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}文档]。
