在您希望启用项目监控的每个项目中安装*项目监视器*。

. 单击 *☰ > 集群管理*。
. 在*集群*页面上，转到您希望启用监控的集群，然后单击*探索*。
. 在左侧导航栏中单击menu:监控[项目监视器]。然后在右上角单击*创建*。
+
image::project-monitors.png[项目监视器]

. 从下拉菜单中选择您的项目，然后再次单击*创建*。
+
image::create-project-monitors.png[创建项目监视器]
