[#_prometheusrule]
== PrometheusRule

PrometheusRule包含您通常会放置在 https://prometheus.io/docs/prometheus/latest/configuration/recording_rules/[Prometheus规则文件]中的告警和记录规则。

对于更细粒度的方法，Prometheus资源上的`ruleSelector`字段可以根据附加到PrometheusRules资源的标签选择应加载到Prometheus上的PrometheusRules。

请查看 https://prometheus-operator.dev/docs/developer/alerting/[Prometheus Operator文档]以获取示例PrometheusRule。
