[#_about_the_prometheus_custom_resource]
== 关于 Prometheus 自定义资源

Prometheus CR 定义了期望的 Prometheus 部署。Prometheus Operator 观察 Prometheus CR。当 CR 发生变化时，Prometheus Operator 会根据 CR 配置创建 `prometheus-rancher-monitoring-prometheus`，即 Prometheus 部署。

Prometheus CR 指定了规则等详细信息，以及连接到 Prometheus 的 Alertmanagers。Rancher 为您构建此 CR。

监控 V2 每个集群仅支持一个 Prometheus。但是，如果您想将监控限制在某些名称空间，您可能需要编辑 Prometheus CR。
