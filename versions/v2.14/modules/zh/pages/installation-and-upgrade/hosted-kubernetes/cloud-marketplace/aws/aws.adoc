[#_overview]
== 概述

Rancher 提供与 AWS Marketplace 的集成，允许用户购买 SUSE 的支持合同。此集成使您能够轻松调整支持需求，因为您开始支持更多集群。

[#_limitations]
== 局限性

* 您必须运行 Rancher v2.6.7 或更高版本。
* Rancher 必须在启用额外指标的情况下部署。
* Rancher 必须安装在 EKS 集群上。
* 您必须通过 AWS Marketplace 购买至少一个 Rancher 支持的授权。
* 您可能需要额外的设置来支持代理/隔离环境的使用案例。有关更多信息，请参见 {xref-aws-cloud-marketplace-adapter-reqs}[先决条件]。

[#_how_to_use]
== 如何使用

. 完成 {xref-aws-cloud-marketplace-adapter-reqs}[先决步骤]。
. {xref-aws-cloud-marketplace-install-adapter}[安装 CSP 适配器]。

[#_faq]
== 常见问题

*我可以稍后为更多节点购买支持吗？*

可以。只需访问您最初用于购买支持的 AWS Marketplace 条目，并增加授权数量。

*我可以在同一 AWS 账户中使用多个 Rancher 实例吗？*

可以。然而，Rancher 安装的每个集群都需要遵守先决条件。

此外，请记住，给定的授权一次只能被一个 Rancher 管理服务器使用。
