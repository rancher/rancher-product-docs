[CAUTION]
====

这些指南的目的是快速启动一个沙盒，供您评估 Rancher。这些指南不适用于生产环境。有关全面的设置说明，请参见 {xref-install-upgrade}[安装]。
====


使用文档的这一部分来快速启动您的 Rancher 2.x 部署和测试。它包含简单的 Rancher 设置和一些常见用例的说明。我们计划在未来向这一部分添加更多内容。

我们有以下快速入门指南：

* {xref-qs-deploy-rancher}[部署 Rancher 服务器]：使用对您来说最方便的方法开始运行 Rancher。
* {xref-qs-deploy-workloads}[部署工作负载]：部署一个简单的 https://kubernetes.io/docs/concepts/workloads/[工作负载] 并将其暴露，让您可以从集群外部访问。
