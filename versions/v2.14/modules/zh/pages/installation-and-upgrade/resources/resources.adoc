[#_docker_installations]
== Docker 安装

xref:[单节点 Docker 安装]适用于希望试用 Rancher 的用户。您可以使用 `docker run` 命令在单个节点上安装 Rancher 服务器组件，而不是在使用 Helm 的 Kubernetes 集群上运行。

由于只有一个节点和一个 Docker 容器，如果节点出现故障，其他节点上将没有 etcd 数据的副本，您将丢失所有 Rancher 服务器的数据。

[#_air_gapped_installations]
== 隔离的安装

按照 {xref-air-gap}[这些步骤]在隔离的环境中安装 Rancher 服务器。

隔离环境可能是指 Rancher 服务器将在离线状态下安装，位于防火墙后面或代理后面。
