[#_air_gapped_installations]
== 隔离的安装

按照{xref-air-gap}[这些步骤]在隔离的环境中安装Rancher服务器。

隔离环境可能是指 Rancher 服务器将在离线状态下安装，位于防火墙后面或代理后面。

[#_docker_installations]
== Docker安装

xref:[单节点Docker安装]适用于希望测试Rancher的Rancher用户。您可以使用`docker run`命令在单个节点上安装Rancher服务器组件，而不是在Kubernetes集群上使用Helm。

Docker安装仅适用于开发和测试环境。

由于只有一个节点和一个Docker容器，如果节点出现故障，其他节点上将没有etcd数据的副本，您将丢失Rancher服务器的所有数据。

Rancher备份操作员可用于将Rancher从单个Docker容器安装迁移到高可用性Kubernetes集群上的安装。有关详细信息，请参阅关于{xref-migrate-cluster}[将Rancher迁移到新集群的文档。]
