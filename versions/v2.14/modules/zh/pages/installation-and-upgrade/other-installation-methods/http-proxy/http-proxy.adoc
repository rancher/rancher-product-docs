在许多企业环境中，运行在本地的服务器或虚拟机没有直接的互联网访问，但出于安全原因必须通过HTTP(S)代理连接到外部服务。本教程逐步展示如何在这样的环境中设置高可用性的Rancher安装。

另外，也可以在完全隔离的环境中安装Rancher，无需任何互联网访问。该过程在{xref-air-gap}[Rancher文档]中有详细描述。
