https://github.com/rancher/charts[图表]储存库包含所有所需的 Helm 目录项，用于监控、日志记录、警报和 Istio 等功能。

在 Rancher 的隔离安装中，您需要配置 Rancher 以使用本地系统图表的副本。本节描述如何通过 CLI 标志使用本地系统图表。

[#_using_local_system_charts]
== 使用本地系统图表

本地副本的 `system-charts` 已打包到 `rancher/rancher` 容器中。要在隔离安装中使用这些功能，您需要运行 Rancher 安装命令，并添加一个额外的环境变量 `CATTLE_SYSTEM_CATALOG=bundled`，该变量告诉 Rancher 使用本地系统图表的副本，而不是尝试从 GitHub 获取它们。

带有捆绑 `system-charts` 的 Rancher 安装示例命令已包含在 {xref-air-gap}[隔离安装] 的 Docker 和 Helm 安装说明中。
