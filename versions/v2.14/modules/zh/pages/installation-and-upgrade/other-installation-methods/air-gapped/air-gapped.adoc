本节介绍如何在隔离环境中使用 Helm CLI 安装 Rancher 服务器。隔离环境可能是指 Rancher 服务器将在离线状态下安装，位于防火墙后面或代理后面。

安装步骤因 Rancher 是安装在 K3s Kubernetes 集群还是单个 Docker 容器而异。

有关每个安装选项的更多信息，请参阅 {xref-install-upgrade}[此页面。]

在整个安装说明中，将为每个安装选项提供 _标签_。

[NOTE]
.重要说明：
====

如果您按照 Docker 安装指南安装 Rancher，则没有升级路径可以将您的 Docker 安装迁移到 Kubernetes 安装。
====


[#_installation_outline]
== 安装大纲

. {xref-air-gap-private-reg}[设置基础设施和私有注册表]
. {xref-air-gap-publish-images}[收集并发布镜像到您的私有注册表]
. {xref-air-gap-install-k8s}[设置 Kubernetes 集群（Docker 安装可跳过此步骤）]
. {xref-air-gap-rancher-ha}[安装 Rancher]

== 升级

要在隔离环境中使用 Helm CLI 升级 Rancher，请按照 {xref-rancher-upgrades}[此流程操作。]
