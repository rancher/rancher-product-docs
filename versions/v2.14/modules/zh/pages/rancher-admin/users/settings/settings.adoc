在Rancher中，每个用户都有与其登录相关的一些设置：个人偏好、API 密钥等。您可以在*用户设置*菜单中配置这些设置。您可以通过点击位于主菜单中的头像来打开此菜单。

image::user-settings.png[用户设置菜单]

可用的用户设置包括：

* {xref-user-api-keys}[API 和密钥]：如果您想以编程方式与Rancher交互，您需要一个API密钥。请按照本节中的说明获取密钥。
* {xref-cloud-credentials}[云凭据]：管理机器池使用的云凭据，以{xref-launch-k8s-with-rancher}[为集群配置节点]。
* {xref-user-preferences}[偏好设置]：设置 Rancher UI 的外观偏好。
* 注销:结束您的用户会话。
