您可以设置偏好以个性化您的Rancher体验。要更改偏好设置：

. 单击右上角的用户头像。
. 单击*偏好设置*。

[#_language]
== 语言

选择Rancher UI的语言。选项包括：

* English
* 简体中文（Simplified Chinese）

[#_theme]
== 主题

选择Rancher UI的背景颜色。如果您选择*自动*，背景颜色将在晚上6点从浅色变为深色，然后在早上6点变回浅色。

[#_login_landing_page]
== 登录着陆页

选择登录后显示的页面。选项包括：

* 主页。
* 您上次访问的区域。
* 您选择的特定集群。

[#_display_settings]
== 显示设置

选择某些信息的显示方式：

* 日期格式
* 时间格式
* 每页表格行数
* 侧边菜单中显示的集群数量

[#_confirmation_setting]
== 确认设置

选择在缩减节点池时是否询问确认。

[#_advanced_features]
== 高级功能

* 启用“在API中查看”。
* 显示由Rancher管理的系统名称空间（不用于编辑或删除）。
* 启用深色/浅色主题的快捷键切换（shift+T）。
* 隐藏所有类型描述。
* 启用扩展开发者功能。

[#_yaml_editor_key_mapping]
== YAML编辑器键映射

* 普通人（Sublime）
* Emacs
* Vim

[#_helm_charts]
== Helm 图表

选择是否仅显示已发布的Helm图表，或同时包括预发布版本。如果一个版本符合 https://semver.org/#spec-item-9[规范]，该规范由 https://semver.org/[语义化版本2.0.0]定义，则被视为预发布版本。例如，版本为`0.1.3-dev.12ab4f`的Helm图表需要选择`Include Prerelease Versions`才能显示。
