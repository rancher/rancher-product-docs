[CAUTION]
====
在 ARM64 混合架构平台上运行目前是一个实验性功能，尚未在 Rancher 中正式支持。因此，我们不建议在生产环境中使用基于 ARM64 混合架构的节点。
====

在使用 ARM64 平台时可用的选项如下：

* 创建自定义集群并添加基于 ARM64 的节点
 ** Kubernetes 集群版本必须为 1.12 或更高
* 导入包含基于 ARM64 的节点的集群
 ** Kubernetes 集群版本必须为 1.12 或更高

根据您的集群配置，请参考 {xref-rke2-config}[RKE2 集群配置选项] 或 {xref-k3s-config}[K3s 集群配置选项] 以获取更多信息。

以下功能尚未经过测试：

* 监控、警报、通知、管道和日志记录
* 从目录中启动 APP
