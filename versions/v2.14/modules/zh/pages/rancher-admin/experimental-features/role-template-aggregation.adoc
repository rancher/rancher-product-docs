[CAUTION]
====
角色模板聚合是v2.13中的一项实验性功能，它改变了用于角色模板、集群角色模板绑定和项目角色模板绑定的RBAC架构。**不支持生产环境**。在v2.13和v2.14之间可能会发生重大变化。
====

角色模板聚合使用Kubernetes功能 https://kubernetes.io/docs/reference/access-authn-authz/rbac/#aggregated-clusterroles[聚合集群角色]来实现角色模板、集群角色模板绑定和项目角色模板绑定。新架构导致Rancher集群和下游集群中的RBAC对象（角色、角色绑定、集群角色和集群角色绑定）净减少。

有关该功能如何改善可伸缩性和性能的更多信息，请参见 https://www.suse.com/c/rancher_blog/fewer-bindings-more-power-ranchers-rbac-boost-for-enhanced-performance-and-scalability/[Rancher博客文章]。

|===
| 环境变量键 | 默认值 | 说明

| `aggregated-roletemplates`
| `false`
| [测试版] 使角色模板使用聚合生成RBAC角色。
|===

此功能标志的值在安装时被锁定，在用户界面中显示为功能标志旁边的锁定符号。这意味着该功能只能在Rancher的首次安装时设置。之后，尝试修改该值将被拒绝。
