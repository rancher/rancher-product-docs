{xref-fleet}[Fleet 的持续交付] 随 Rancher 预装，且无法完全禁用。但是，可以使用 `continuous-delivery` 功能标志禁用 Fleet 中用于 GitOps 持续交付的功能。

要启用或禁用此功能，请参阅{xref-experimental-features}[关于启用实验性功能的主页]上的说明。

|===
| 环境变量键 | 默认值 | 说明

| `continuous-delivery`
| `true`
| 该标志用于禁用 Fleet 中用于 GitOps 持续交付的功能。
|===

如果在 Rancher v2.5.x 中禁用了 Fleet，则在升级到 v2.6.x 时将重新启用。仅可以禁用 Fleet 的持续交付部分。当 `continuous-delivery` 被禁用时，`gitjob` 部署将不再部署到 Rancher 服务器的本地群集中，且 `continuous-delivery` 不会显示在 Rancher UI 中。
