Rancher仪表板提供动态通知和系统警报，以确保您了解与您的部署相关的最新更新、生命周期事件和状态。

[#_dynamic_content]
== 动态内容

Rancher获取动态内容，包括有关新版本、维护结束（EOM）或终止服务（EOL）状态以及其他重要公告的通知。

动态内容通过Rancher仪表板中的三个区域传递给用户。

* *主页通知横幅*：在主页顶部显示的动态横幅，用于高优先级公告。
* *主页链接下方的区域*：主页上专门的区域，位于*链接*部分下方，显示相关更新和资源。
* *<<_what_is_the_notification_center,通知中心>>*：所有通知的集中枢纽，包括系统警报、资源和公告。

要与这些通知互动，请点击列出资源上的*了解更多*。您也可以关闭通知。

[#_what_is_the_notification_center]
== 什么是通知中心？

通知中心作为系统生成的警报和动态公告的中央储存库。您可以通过点击Rancher仪表板右上角的铃铛图标访问它。

通知按严重性和类型分类：

* *错误*表示高严重性问题。
* *警告*表示中等严重性问题。
* *信息*提供较低严重性的常规更新。
* *任务*表示某个过程或行动正在进行中。
* *成功*确认某个过程或行动已成功完成。

例如，当有新版本的 Rancher 可用时，您可能会收到通知，或者直接链接到 Rancher 发布说明。

您可以使用上下箭头键轻松浏览通知。某些通知可能还包括相关的操作，允许您立即做出回应。
