[#_what_can_i_expect_when_a_feature_is_marked_for_deprecation]
== 当某个功能被标记为弃用时，我可以期待什么？

在功能被标记为 "弃用" 的版本中，它仍然可用并受到支持，允许按照通常的程序进行升级。一旦升级，用户/管理员应开始计划在升级到标记为去除的版本之前，逐步放弃弃用的功能。对于新部署，建议不要使用弃用的功能。
