`kubectl` 是一个用于对 Kubernetes 集群运行命令的 CLI 工具。它是 Rancher 2.x 中许多维护和管理任务所必需的。

[#_installation]
== 安装

请参见 https://kubernetes.io/docs/tasks/tools/install-kubectl/[kubectl 安装] 以获取您操作系统上的安装信息。

[#_configuration]
== 配置

当您使用 RKE2/K3s 创建 Kubernetes 集群时，Kubeconfig 文件存储在 `/etc/rancher/rke2/rke2.yaml` 或 `/etc/rancher/k3s/k3s.yaml`，具体取决于您选择的发行版。这些文件用于配置对 Kubernetes 集群的访问。

测试您与 `kubectl` 的连通性，看看是否可以获取节点列表。

----
kubectl get nodes
 NAME                          STATUS    ROLES                      AGE       VERSION

165.227.116.167               Ready     controlplane,etcd,worker   11m       v1.10.1
165.227.127.226               Ready     controlplane,etcd,worker   11m       v1.10.1
----
