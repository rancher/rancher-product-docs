[IMPORTANT]
====
加密密钥轮换默认启用，无法禁用。
====

要在 Rancher 用户界面中轮换密钥：

. 单击 *☰ > 集群管理*。
. 在屏幕最右侧靠近您选择的集群的位置，选择*⋮ > 轮换加密密钥*：
+
image::rke2-encryption-key.png[加密密钥轮换]

NOTE: 有关RKE2秘密加密配置的更多信息，请参见 https://documentation.suse.com/cloudnative/rke2/latest/en/security/secrets_encryption.html[RKE2文档]。
