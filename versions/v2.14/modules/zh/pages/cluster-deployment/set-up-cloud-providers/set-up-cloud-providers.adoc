_云提供商_是Kubernetes中的一个模块，提供管理节点、负载均衡器和网络路由的接口。

当在Rancher中设置云提供商时，如果您使用的云提供商支持此类自动化，Rancher服务器可以在启动Kubernetes定义时自动配置新的节点、负载均衡器或持久存储设备。

如果您配置的云提供商节点集群不满足先决条件，则您的集群将无法正确配置。

默认情况下，*云提供商*选项设置为`None`。

可以启用以下云提供商：

* 亚马逊
* Azure
* GCE（谷歌计算引擎）
* vSphere

[#_setting_up_a_custom_cloud_provider]
== 设置自定义云提供商

如果您想配置任何Kubernetes云提供商，则`Custom`云提供商可用。

对于自定义云提供商选项，您可以参考 https://rancher.com/docs/rke/latest/en/config-options/cloud-providers/[RKE文档]，了解如何为您的特定云提供商编辑yaml文件。有一些特定的云提供商具有更详细的配置：

* https://rke.docs.rancher.com/config-options/cloud-providers/vsphere[vSphere]
* https://rancher.com/docs/rke/latest/en/config-options/cloud-providers/openstack/[OpenStack]
