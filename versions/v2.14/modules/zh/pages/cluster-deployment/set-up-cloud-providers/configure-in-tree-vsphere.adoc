要设置内置的 VMware vSphere 云提供商，请在 Rancher 中创建 vSphere 集群时遵循以下步骤：

. 将 *云提供商* 选项设置为 `Custom` 或 `Custom (In-Tree)`。
. 点击 *编辑 YAML*
. 将以下结构插入到预填充的集群 YAML 中。该结构必须放置在 `rancher_kubernetes_engine_config` 下。请注意，`name` _必须_ 设置为 `vsphere`。
+
[,yaml]
----
 rancher_kubernetes_engine_config:
   cloud_provider:
       name: vsphere
       vsphereCloudProvider:
           [Insert provider configuration]
----

Rancher 使用 RKE（Rancher Kubernetes 引擎）来部署 Kubernetes 集群。请参阅 https://rancher.com/docs/rke/latest/en/config-options/cloud-providers/vsphere/config-reference/[RKE 文档中的 vSphere 配置参考]，以了解 `vsphereCloudProvider` 指令属性的详细信息。
