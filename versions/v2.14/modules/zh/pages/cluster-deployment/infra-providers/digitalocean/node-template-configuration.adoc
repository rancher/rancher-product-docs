[#_droplet_options]
== Droplet 选项

*Droplet 选项* 用于为您的集群配置地理区域和规格。

[#_docker_daemon]
== Docker 守护程序

如果您使用 Docker， https://docs.docker.com/engine/docker-overview/#the-docker-daemon[Docker 守护程序] 的配置选项包括：

* *标签：*有关标签的信息，请参阅 https://docs.docker.com/config/labels-custom-metadata/[Docker 对象标签文档。]
* *Docker 引擎安装 URL：*确定将在实例上安装哪个 Docker 版本。
* *注册表镜像：*Docker 守护程序将使用的 Docker 注册表镜像
* *其他高级选项：*请参阅 https://docs.docker.com/engine/reference/commandline/dockerd/[Docker 守护程序选项参考]。
