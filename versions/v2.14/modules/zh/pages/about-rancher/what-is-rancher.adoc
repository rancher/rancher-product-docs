:sidebar_label: 什么是 Rancher？
:slug: /

Rancher 是一个 Kubernetes 管理工具，可在任何位置和任何提供商上部署和运行群集。

Rancher 可以从托管服务提供商处置备 Kubernetes、置备计算节点并在节点上安装 Kubernetes，或导入任意环境中已运行的 Kubernetes 群集。

Rancher 在 Kubernetes 基础上大幅提升价值：首先集中管理所有群集的身份验证和基于角色的访问控制（RBAC），让全局管理员可从单一位置管控群集访问权限。

它随后支持对群集及其资源进行精细化监控和提供相关警告，将日志发送至外部服务提供商，并通过应用目录直接与 Helm 集成。如果您有外部 CI/CD 系统，可以将其接入 Rancher，但如果没有，Rancher 甚至包括 Fleet 来帮助您自动部署和升级工作负载。

Rancher 是一个 _完整_ 的 Kubernetes 容器管理平台，为您提供在任意环境成功运行 Kubernetes 的工具。
