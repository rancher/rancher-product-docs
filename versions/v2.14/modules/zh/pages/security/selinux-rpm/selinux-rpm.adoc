https://en.wikipedia.org/wiki/Security-Enhanced_Linux[安全增强Linux (SELinux)] 是对Linux的安全增强。

由红帽公司开发，它是在Linux上实现强制访问控制（MAC）的方案。强制访问控制允许系统管理员定义应用程序和用户如何访问不同的资源，如文件、设备、网络和进程间通信。SELinux还通过默认限制操作系统来增强安全性。

在历史上被政府机构使用后，SELinux现在已成为行业标准，并在SUSE发行版（如SUSE Linux Enterprise 16）、openSUSE发行版（如openSUSE Tumbleweed）和基于RHEL的操作系统（如Rocky Linux）中默认启用。要检查SELinux是否在您的系统上启用并强制执行，请使用 `getenforce`：

----
# getenforce
Enforcing
----
