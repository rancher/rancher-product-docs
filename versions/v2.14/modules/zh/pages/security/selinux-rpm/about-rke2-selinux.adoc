`rke2-selinux` 为 RKE2 提供策略。当 RKE2 安装脚本检测到运行在基于 RPM 的发行版上时，会自动安装。

`rke2-selinux` 的 GitHub 储存库位于 https://github.com/rancher/rke2-selinux[这里。]

有关在启用 SELinux 的主机上安装 RKE2 的更多信息，请参见 https://documentation.suse.com/cloudnative/rke2/latest/en/install/methods.html#rpm[RKE2 文档。]
