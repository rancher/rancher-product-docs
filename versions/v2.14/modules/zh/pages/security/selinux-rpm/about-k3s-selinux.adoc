`k3s-selinux` 为 K3s 提供策略。当 K3s 安装脚本检测到它运行在基于 RPM 的发行版上时，会自动安装。

`k3s-selinux` 的 GitHub 储存库在 https://github.com/rancher/k3s-selinux[这里]。

有关在启用 SELinux 的主机上安装 K3s 的更多信息，请参见 https://documentation.suse.com/cloudnative/k3s/latest/en/advanced.html#_selinux_support[K3s 文档]。
