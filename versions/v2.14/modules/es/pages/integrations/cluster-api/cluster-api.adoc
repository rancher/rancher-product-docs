https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}] es un https://kubernetes.io/docs/concepts/extend-kubernetes/operator/#operators-in-kubernetes[Operador de Kubernetes] que gestiona el ciclo de vida de los clústeres de Kubernetes provisionados, proporcionando integración entre tu Cluster API (CAPI) y Rancher. Con Rancher Turtles, puedes:

* Importar clústeres CAPI en Rancher, instalando el Agente de Clúster de Rancher en clústeres provisionados por CAPI.
* Gestionar el https://cluster-api-operator.sigs.k8s.io/[Operador CAPI] utilizando el recurso {url-turtles-capiprovider}[CAPIProvider].

La sección {xref-turtles-overview}[Visión general] describe las opciones de instalación, la arquitectura de Rancher Turtles y una breve demostración. Para más detalles, consulta la https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name} documentación].
