Fleet orquesta y gestiona la entrega continua de aplicaciones a través del sistema de suministro para flotas de clústeres. Fleet organiza el sistema de suministro para ayudar a los equipos a entregar con confianza y seguridad de manera oportuna utilizando GitOps como un modelo operativo seguro.

[#__fleet_product_name_with_rancher]
== {fleet-product-name} con Rancher

Muchos usuarios a menudo gestionan más de 10 clústeres a la vez. Dada la proliferación de clústeres, la entrega continua es una parte importante de Rancher. Fleet garantiza una experiencia de entrega continua fiable utilizando GitOps, que es un modelo operativo seguro y cada vez más común.

[#_when_should_i_use_fleet_product_name]
=== ¿Cuándo debo usar {fleet-product-name}?

* Necesito desplegar mi pila de monitorización (por ejemplo, Grafana, Prometheus) a través de regiones geográficas, cada una con diferentes políticas de retención.
* Soy un operador de plataforma y quiero aprovisionar clústeres con todos los componentes utilizando un modelo operativo escalable y seguro (GitOps).
* Soy un desarrollador de aplicaciones y quiero que mis últimos cambios vayan automáticamente a mi entorno de desarrollo.

[#__fleet_product_name_with_rancher_prime]
== {fleet-product-name} con {rancher-prime}

Fleet ya está profundamente integrado como la herramienta de entrega continua y motor de GitOps en Rancher.
