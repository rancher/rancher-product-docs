En muchos entornos empresariales, los servidores o máquinas virtuales que se ejecutan en las instalaciones no tienen acceso directo a Internet, pero deben conectarse a servicios externos a través de un proxy HTTP(S) por razones de seguridad. Este tutorial muestra paso a paso cómo configurar una instalación de Rancher altamente disponible en un entorno así.

Alternativamente, también es posible configurar Rancher en un entorno completamente aislado sin acceso a Internet. Este proceso se describe en detalle en la {xref-air-gap}[documentación de Rancher].
