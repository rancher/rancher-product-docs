El repositorio https://github.com/rancher/charts[Gráficos] contiene todos los elementos del catálogo de Helm requeridos para características como monitoreo, registro, alertas e Istio.

En una instalación en entorno aislado de Rancher, necesitarás configurar Rancher para usar una copia local de los gráficos del sistema. Esta sección describe cómo usar los gráficos del sistema local utilizando una opción de CLI.

[#_using_local_system_charts]
== Uso de Gráficos del Sistema Local

Una copia local de `system-charts` ha sido empaquetada en el contenedor `rancher/rancher`. Para poder usar estas características en una instalación en entorno aislado, necesitarás ejecutar el comando de instalación de Rancher con una variable de entorno adicional, `CATTLE_SYSTEM_CATALOG=bundled`, que le indica a Rancher que use la copia local de los gráficos en lugar de intentar obtenerlos de GitHub.

Se incluyen comandos de ejemplo para una instalación de Rancher con un `system-charts` empaquetado en las instrucciones [de instalación en entorno aislado] de {xref-air-gap} para instalaciones de Docker y Helm.
