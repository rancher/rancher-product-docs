[CAUTION]
====

El objetivo de estas guías es lanzar rápidamente un entorno aislado que puedas utilizar para evaluar Rancher. Estas guías no están destinadas a entornos de producción. Para instrucciones de configuración completas, consulta {xref-install-upgrade}[Instalación].
====


Utiliza esta sección de la documentación para iniciar tu despliegue y pruebas de Rancher 2.x. Contiene instrucciones para una configuración sencilla de Rancher y algunos casos de uso comunes. Planeamos añadir más contenido a esta sección en el futuro.

Tenemos guías de inicio rápido para:

* {xref-qs-deploy-rancher}[Despliegue del Servidor Rancher]: Empieza a ejecutar Rancher utilizando el método que te resulte más conveniente.
* {xref-qs-deploy-workloads}[Despliegue de Cargas de Trabajo]: Despliega una https://kubernetes.io/docs/concepts/workloads/[carga de trabajo] sencilla y expónla, permitiéndote acceder a ella desde fuera del clúster.
