El propósito de esta sección es consolidar las mejores prácticas para las implementaciones de Rancher. Esto también incluye recomendaciones para tecnologías relacionadas, como Kubernetes, Docker, contenedores y más. El objetivo es mejorar el resultado de una implementación de Rancher utilizando la experiencia operativa de Rancher y sus clientes.

Si tienes alguna pregunta sobre cómo podrían aplicarse a tu caso de uso, por favor contacta a tu Gerente de Éxito del Cliente o Soporte.

Utiliza la barra de navegación a la izquierda para encontrar las mejores prácticas actuales para gestionar y desplegar el Servidor Rancher.

Para más orientación sobre las mejores prácticas, puedes consultar estos recursos:

* https://www.suse.com/c/rancherblog/[Rancher Blog]
* https://forums.rancher.com/[Foro de Rancher]
