Esta guía contiene nuestras recomendaciones para ejecutar el servidor Rancher y está destinada a ser utilizada en situaciones en las que Rancher gestiona clústeres de Kubernetes en sentido descendente.

== Arquitectura e Infraestructura Recomendadas

Consulta esta {xref-tips-for-running-rancher}[guía] para nuestro consejo general sobre la configuración del servidor Rancher para una instalación en producción.

[#_deployment_strategies]
== Estrategias de distribución

Esta {xref-deployment-strategy}[guía] está diseñada para ayudarle a elegir si una estrategia de despliegue regional o una estrategia de despliegue hub-and-spoke es mejor para un servidor Rancher que gestiona clústeres de Kubernetes en sentido descendente.
