Rancher ofrece integración con marketplaces en la nube para adquirir fácilmente soporte para instalaciones alojadas en ciertos proveedores de nube. Además, esta integración también permite generar un paquete supportconfig que se puede proporcionar al soporte de Rancher.

Esta integración solo es compatible con AWS.
