Esta sección contiene los requisitos para Helm, que es la herramienta utilizada para instalar Rancher en un clúster de Kubernetes de alta disponibilidad.

[#_identifying_the_proper_helm_v3_version]
== Identificación de la versión adecuada de Helm v3

Selecciona cualquier versión de Helm v3 que sea oficialmente compatible con el rango de versiones de Kubernetes que estás utilizando de nuestra https://www.suse.com/suse-rancher/support-matrix/all-supported-versions[Matriz de Soporte de Rancher].

Para aplicar esta regla, puede que necesites consultar dos recursos externos:

* *Compatibilidad de versiones de Helm:* Consulta la https://helm.sh/docs/topics/version_skew/[Política de Soporte de Versiones de Helm] y selecciona la versión que coincida con la regla para tu versión menor de Rancher.
* *Rango de Soporte de Kubernetes de Rancher:* Utiliza la https://www.suse.com/suse-rancher/support-matrix/all-supported-versions[Matriz de Soporte de Rancher] para identificar las versiones de Kubernetes soportadas por tu versión menor de Rancher objetivo.

[#_example]
=== Ejemplo

* *Escenario:* Estás apuntando a Rancher v2.11.4, que soporta las versiones de Kubernetes de 1.30 a 1.32.
* *Aplicación:* Nuestra regla requiere una versión de Helm que soporte este rango. Puedes verificar esto comprobando la compatibilidad de la versión de Helm con la versión más alta en el rango, Kubernetes v1.32.
* *Resultado:* Descubres que tanto Helm v3.17 como Helm v3.18 soportan el rango de Kubernetes v1.30-v1.32.
* Aunque ambos funcionan, recomendamos Helm v3.18 porque es la versión menor de Helm más nueva que se superpone con el rango de Kubernetes soportado.

[#_additional_notes]
== Notas adicionales

* Se requiere Helm v3.2.x o superior para instalar o actualizar Rancher v2.5.
* Al utilizar herramientas que ejecutan comandos de Helm por ti (como Terraform), debes asegurarte de que estén configuradas para usar la versión correcta de Helm.
