Para más detalles sobre DigitalOcean, Droplets, consulta la https://docs.digitalocean.com/products/compute/[documentación oficial].

[#_region]
== Región

Configura la https://docs.digitalocean.com/glossary/region/[región] donde se crean los Droplets.

[#_size]
== Tamaño

Configura el https://docs.digitalocean.com/products/droplets/resources/choose-plan/[tamaño] de los Droplets.

[#_os_image]
== Imagen del SO

Configura la https://docs.digitalocean.com/products/images/[imagen] del sistema operativo de la que se crean los Droplets.

[#_monitoring]
== Monitorización

Habilita el agente de DigitalOcean para un https://docs.digitalocean.com/products/monitoring/[monitoreo] adicional.

[#_ipv6]
== IPv6

Habilita IPv6 para los Droplets.

Para más información, consulta la https://docs.digitalocean.com/products/networking/ipv6[documentación de IPv6 de DigitalOcean].

[#_private_networking]
== Red privada

Habilita la red privada para los Droplets.

[#_droplet_tags]
== Etiquetas de Droplet

Aplica una etiqueta a un Droplet. Las etiquetas solo pueden contener letras, números, dos puntos, guiones y guiones bajos. Por ejemplo, `my_server`.
