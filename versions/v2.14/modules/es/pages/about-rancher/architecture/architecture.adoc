Esta sección se centra en el {xref-arch-components}[servidor Rancher y sus componentes] y en cómo {xref-arch-com-ds-cluster}[Rancher se comunica con los clusters de Kubernetes en sentido descendente].

Para información sobre las diferentes formas en que se puede instalar Rancher, consulta la {xref-install-upgrade}#_overview_of_installation_options[sección de visión general de las opciones de instalación.]

Para una lista de las principales características del servidor API de Rancher, consulta la {xref-rancher-overview}[sección de visión general.]

Para orientación sobre cómo configurar la infraestructura subyacente para el servidor Rancher, consulta las {xref-arch-rec}[recomendaciones arquitectónicas.]

[NOTE]
====

Esta sección asume un conocimiento básico de Docker y Kubernetes. Para una breve explicación de cómo los componentes de Kubernetes trabajan juntos, consulta la {xref-k8s-concepts}[página de conceptos].
