El https://kubernetes.io/docs/tasks/run-application/horizontal-pod-autoscale/[Escalador horizontal de pods] (HPA) es una característica de Kubernetes que permite configurar tu clúster para escalar automáticamente los servicios que se están ejecutando hacia arriba o hacia abajo.

Rancher proporciona algunas características adicionales para ayudar a gestionar los HPAs, dependiendo de la versión de Rancher.

Puedes crear, gestionar y eliminar HPAs utilizando la interfaz de usuario de Rancher. Solo admite HPA en la `autoscaling/v2beta2` API.

[#_managing_hpas]
== Gestión de HPAs

La forma en que gestionas los HPAs es diferente según tu versión de la API de Kubernetes:

* *Para la versión de API de Kubernetes autoscaling/V2beta1:* Esta versión de la API de Kubernetes te permite escalar automáticamente tus pods en función de la utilización de CPU y memoria de tu aplicación.
* *Para la versión de API de Kubernetes autoscaling/V2beta2:* Esta versión de la API de Kubernetes te permite escalar automáticamente tus pods en función de la utilización de CPU y memoria, además de métricas personalizadas.

Puedes crear, gestionar y eliminar HPAs utilizando la interfaz de usuario de Rancher. Desde la interfaz de usuario de Rancher puedes configurar el HPA para escalar en función de la utilización de CPU y memoria. Para más información, consulta {xref-hpas-with-ui}[Gestionar HPAs con la GUI de Rancher]. Para escalar el HPA en función de métricas personalizadas, aún necesitas utilizar `kubectl`. Para más información, consulta {xref-hpas-with-kubectl}#_configuring_hpa_to_scale_using_custom_metrics_with_prometheus[Configurando el HPA para escalar utilizando métricas personalizadas con Prometheus].

Los clústeres creados en Rancher v2.0.7 y superiores tienen automáticamente todos los requisitos necesarios (metrics-server y configuración del clúster de Kubernetes) para utilizar el HPA.

[#_testing_hpas_with_a_service_deployment]
== Pruebas de HPAs con un despliegue de servicio

Puedes ver el número actual de réplicas de tu HPA yendo a tu proyecto y haciendo clic en menu:Recursos[HPA]. Para más información, consulta {xref-hpas-with-ui}[Obtener métricas y estado del HPA].

También puedes utilizar `kubectl` para obtener el estado de los HPAs que pruebas con tu herramienta de pruebas de carga. Para más información, consulta {xref-hpas-test}[Pruebas de HPAs con kubectl].
