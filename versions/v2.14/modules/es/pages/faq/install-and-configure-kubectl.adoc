`kubectl` es una utilidad CLI para ejecutar comandos contra clústeres de Kubernetes. Es necesaria para muchas tareas de mantenimiento y administrativas en Rancher 2.x.

[#_installation]
== Instalación

Consulta https://kubernetes.io/docs/tasks/tools/install-kubectl/[Instalación de kubectl] para la instalación en tu sistema operativo.

[#_configuration]
== Configuración

Cuando creas un clúster de Kubernetes con RKE2/K3s, el archivo Kubeconfig se almacena en `/etc/rancher/rke2/rke2.yaml` o `/etc/rancher/k3s/k3s.yaml` dependiendo de la distribución elegida. Estos archivos se utilizan para configurar el acceso al clúster de Kubernetes.

Prueba tu conectividad con `kubectl` y verifica si puedes obtener la lista de nodos.

----
kubectl get nodes

165.227.116.167               Ready     controlplane,etcd,worker   11m       v1.10.1
165.227.127.226               Ready     controlplane,etcd,worker   11m       v1.10.1
----
