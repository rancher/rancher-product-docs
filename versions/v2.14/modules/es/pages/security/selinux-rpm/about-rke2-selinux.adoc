`rke2-selinux` proporciona políticas para RKE2. Se instala automáticamente cuando el guion instalador de RKE2 detecta que se está ejecutando en una distribución basada en RPM.

El `rke2-selinux` repositorio de GitHub está https://github.com/rancher/rke2-selinux[aquí.]

Para más información sobre la instalación de RKE2 en hosts habilitados para SELinux, consulta la https://documentation.suse.com/cloudnative/rke2/latest/en/install/methods.html#rpm[documentación de RKE2.]
