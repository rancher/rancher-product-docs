`k3s-selinux` proporciona políticas para K3s. Se instala automáticamente cuando el guion instalador de K3s detecta que se está ejecutando en una distribución basada en RPM.

El repositorio de GitHub de `k3s-selinux` está https://github.com/rancher/k3s-selinux[aquí].

Para más información sobre la instalación de K3s en hosts habilitados para SELinux, consulta la https://documentation.suse.com/cloudnative/k3s/latest/en/advanced.html#_selinux_support[documentación de K3s].
