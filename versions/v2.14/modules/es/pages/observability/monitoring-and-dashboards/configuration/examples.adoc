[#_prometheusrule]
== PrometheusRule

Una PrometheusRule contiene las reglas de alerta y grabación que normalmente colocarías en un https://prometheus.io/docs/prometheus/latest/configuration/recording_rules/[archivo de reglas de Prometheus].

Para un enfoque más detallado, el campo `ruleSelector` en un recurso de Prometheus puede seleccionar qué PrometheusRules deben ser cargadas en Prometheus, basándose en las etiquetas adjuntas a los recursos de PrometheusRules.

Consulta la https://prometheus-operator.dev/docs/developer/alerting/[documentación del Prometheus Operator] para un ejemplo de PrometheusRule.
