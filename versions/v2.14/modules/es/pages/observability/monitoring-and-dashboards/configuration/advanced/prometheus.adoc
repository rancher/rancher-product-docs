[#_about_the_prometheus_custom_resource]
== Acerca del recurso personalizado de Prometheus

El CR de Prometheus define un despliegue deseado de Prometheus. El Operador de Prometheus observa el CR de Prometheus. Cuando el CR cambia, el Operador de Prometheus crea `prometheus-rancher-monitoring-prometheus`, un despliegue de Prometheus basado en la configuración del CR.

El CR de Prometheus especifica detalles como reglas y qué Alertmanagers están conectados a Prometheus. Rancher construye este CR para usted.

Monitoring V2 solo admite un Prometheus por clúster. Sin embargo, puede que desee editar el CR de Prometheus si quiere limitar la monitorización a ciertos espacios de nombres.
