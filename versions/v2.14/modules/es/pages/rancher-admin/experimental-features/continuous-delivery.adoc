{xref-fleet}[La entrega continua con Fleet] viene preinstalada en Rancher y no se puede desactivar completamente. Sin embargo, la función de Fleet para la entrega continua de GitOps puede desactivarse utilizando la bandera de función `continuous-delivery`.

Para habilitar o deshabilitar esta característica, consulta las instrucciones en {xref-experimental-features}[la página principal sobre habilitar características experimentales.]

|===
| Clave de variable de entorno | Valor por defecto | Descripción

| `continuous-delivery`
| `true`
| Esta bandera desactiva la función de entrega continua de GitOps de Fleet.
|===

Si Fleet se desactivó en Rancher v2.5.x, se activará si Rancher se actualiza a la versión v2.6.x. Solo se puede desactivar la parte de entrega continua de Fleet. Cuando `continuous-delivery` está desactivado, la ampliación de `gitjob` ya no se despliega en el clúster local del servidor de Rancher, y `continuous-delivery` no se muestra en la interfaz de usuario de Rancher.
