Puedes establecer preferencias para personalizar tu experiencia en Rancher. Para cambiar la configuración de preferencias:

. Haz clic en tu avatar de usuario en la esquina superior derecha.
. Haz clic en *Preferencias*.

[#_language]
== Idioma

Elige tu idioma para la interfaz de usuario de Rancher. Las opciones son:

* English
* Chino simplificado (简体中文)

[#_theme]
== Tema

Elige tu color de fondo para la interfaz de usuario de Rancher. Si eliges *Auto*, el color de fondo cambia de claro a oscuro a las 6 PM, y luego vuelve a cambiar a las 6 AM.

[#_login_landing_page]
== Página de inicio de sesión

Elige la página que se muestra después de iniciar sesión. Las opciones son:

* La página de inicio.
* El área que visitaste por última vez.
* Un clúster específico de tu elección.

[#_display_settings]
== Presentar los ajustes

Elige cómo se muestra cierta información:

* Formato de fecha
* Formato de hora
* Filas de tabla por página
* Número de clústeres a mostrar en el menú lateral

[#_confirmation_setting]
== Configuración de confirmación

Elige si pedir confirmación al reducir los grupos de nodos.

[#_advanced_features]
== Funciones avanzadas

* Habilitar "Ver en API".
* Mostrar los espacios de nombres del sistema gestionados por Rancher (no destinados a la edición ni a la eliminación).
* Habilitar el atajo de teclado para alternar entre el tema oscuro/claro (shift+T).
* Ocultar todas las descripciones de tipo.
* Habilitar características para desarrolladores de la extensión.

[#_helm_charts]
== Charts de Helm

Elige si mostrar solo los charts de Helm liberados o incluir también las versiones prerelease. Una versión se considera prerelease si se adhiere a https://semver.org/#spec-item-9[la especificación] definida por https://semver.org/[Semantic Versioning 2.0.0]. Por ejemplo, un chart de Helm con una versión de `0.1.3-dev.12ab4f` requiere que se seleccione `Include Prerelease Versions` para ser mostrado.
