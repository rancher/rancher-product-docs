Dentro de Rancher, cada usuario tiene una serie de ajustes asociados a su inicio de sesión: preferencias personales, claves de API, etc. Puedes configurar estos ajustes eligiendo del menú *Ajustes de usuario*. Puedes abrir este menú haciendo clic en tu avatar, que se encuentra dentro del menú principal.

image::user-settings.png[Menú de Ajustes de Usuario]

Los ajustes de usuario disponibles son:

* {xref-user-api-keys}[API & Keys]: Si deseas interactuar con Rancher de forma programática, necesitas una clave de API. Sigue las instrucciones en esta sección para obtener una clave.
* {xref-cloud-credentials}[Credenciales de Nube]: Gestiona las credenciales de nube utilizadas por los grupos de máquinas para {xref-launch-k8s-with-rancher}[provisionar nodos para clústeres].
* {xref-user-preferences}[Preferencias]: Establece preferencias superficiales para la interfaz de usuario de Rancher.
* Fin de sesión: Finaliza tu sesión de usuario.
