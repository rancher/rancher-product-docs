L'objectif de cette section est de consolider les meilleures pratiques pour les implémentations de Rancher. Cela inclut également des recommandations pour les technologies connexes, telles que Kubernetes, Docker, les conteneurs, et plus encore. L'objectif est d'améliorer le résultat d'une implémentation de Rancher en utilisant l'expérience opérationnelle de Rancher et de ses clients.

Si vous avez des questions sur la manière dont cela pourrait s'appliquer à votre cas d'utilisation, veuillez contacter votre Responsable de la Réussite Client ou le Support.

Utilisez la barre de navigation à gauche pour trouver les meilleures pratiques actuelles pour gérer et déployer le Serveur Rancher.

Pour plus de conseils sur les meilleures pratiques, vous pouvez consulter ces ressources{nbsp}:
