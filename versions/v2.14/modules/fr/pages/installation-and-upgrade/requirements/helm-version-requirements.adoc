Cette section contient les exigences pour Helm, qui est l'outil utilisé pour installer Rancher sur un cluster Kubernetes à haute disponibilité.

[#_identifying_the_proper_helm_v3_version]
== Identification de la version appropriée de Helm v3

Sélectionnez toute version de Helm v3 qui est officiellement compatible avec la plage de versions de Kubernetes que vous utilisez dans notre https://www.suse.com/suse-rancher/support-matrix/all-supported-versions[Matrice de support Rancher].

Pour appliquer cette règle, vous devrez peut-être vous référer à deux ressources externes :

* *Compatibilité des versions de Helm :* Consultez la https://helm.sh/docs/topics/version_skew/[Stratégie de support des versions de Helm] et sélectionnez la version correspondant à la règle pour votre version mineure cible de Rancher.
* *Plage de prise en charge de Kubernetes par Rancher :* Utilisez la https://www.suse.com/suse-rancher/support-matrix/all-supported-versions[Matrice de support Rancher] pour identifier les versions de Kubernetes prises en charge par votre version mineure cible de Rancher.

[#_example]
=== Par exemple :

* *Scénario :* Vous ciblez Rancher v2.11.4, qui prend en charge les versions de Kubernetes 1.30 à 1.32.
* *Application :* Notre règle exige une version de Helm qui prend en charge cette plage. Vous pouvez vérifier cela en vérifiant la compatibilité de la version de Helm avec la version la plus élevée de la plage, Kubernetes v1.32.
* *Résultat :* Vous constatez que les versions Helm v3.17 et Helm v3.18 prennent en charge la plage Kubernetes v1.30-v1.32.
* Bien que les deux fonctionnent, nous recommandons Helm v3.18 car c'est la version mineure de Helm la plus récente qui chevauche la plage de Kubernetes prise en charge.

[#_additional_notes]
== Remarques supplémentaires

* Helm v3.2.x ou supérieur est requis pour installer ou mettre à niveau Rancher v2.5.
* Lorsque vous utilisez des outils qui exécutent des commandes Helm pour vous (comme Terraform), vous devez vous assurer qu'ils sont configurés pour utiliser la bonne version de Helm.
