Rancher propose une intégration avec les marketplaces cloud pour acheter facilement du support pour les installations hébergées sur certains fournisseurs de cloud. De plus, cette intégration permet également de générer un bundle de supportconfig qui peut être fourni au support de Rancher.

Cette intégration ne prend en charge qu'AWS.
