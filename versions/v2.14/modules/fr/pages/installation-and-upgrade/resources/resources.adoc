[#_docker_installations]
== Installations Docker

L'xref:[installation Docker à nœud unique] est destinée aux utilisateurs de Rancher qui souhaitent tester Rancher. Au lieu de fonctionner sur un cluster Kubernetes en utilisant Helm, vous installez le composant serveur Rancher sur un nœud unique en utilisant une `docker run` commande.

Étant donné qu'il n'y a qu'un seul nœud et un seul conteneur Docker, si le nœud tombe en panne, il n'y a pas de copie des données etcd disponibles sur d'autres nœuds et vous perdrez toutes les données de votre serveur Rancher.

[#_air_gapped_installations]
== Installations isolées physiquement

Suivez {xref-air-gap}[ces étapes] pour installer le serveur Rancher dans un environnement isolé physiquement.

Un environnement isolé physiquement pourrait être un endroit où le serveur Rancher sera installé hors ligne, derrière une passerelle de périmètre de sécurité ou un proxy.
