Dans de nombreux environnements d'entreprise, les serveurs ou les machines virtuelles fonctionnant sur site n'ont pas d'accès direct à Internet, mais doivent se connecter à des services externes via un proxy HTTP(S) pour des raisons de sécurité. Ce tutoriel montre étape par étape comment configurer une installation de Rancher hautement disponible dans un tel environnement.

Alternativement, il est également possible de configurer Rancher complètement isolé physiquement sans aucun accès à Internet. Ce processus est décrit en détail dans la {xref-air-gap}[documentation de Rancher].
