[CAUTION]
====

L'objectif de ces guides est de lancer rapidement une sandbox que vous pouvez utiliser pour évaluer Rancher. Ces guides ne sont pas destinés aux environnements de production. Pour des instructions de configuration complètes, voir {xref-install-upgrade}[Installation].
====


Utilisez cette section de la documentation pour démarrer rapidement votre déploiement et tester Rancher 2.x. Elle contient des instructions pour une configuration simple de Rancher et quelques cas d'utilisation courants. Nous prévoyons d'ajouter plus de contenu à cette section à l'avenir.

Nous avons des guides de démarrage rapide pour{nbsp}:

* {xref-qs-deploy-rancher}[Déploiement du serveur Rancher]{nbsp}: Commencez à exécuter Rancher en utilisant la méthode qui vous convient le mieux.
* {xref-qs-deploy-workloads}[Déploiement des charges de travail]{nbsp}: Déployez une simple https://kubernetes.io/docs/concepts/workloads/[charge de travail] et exposez-la, vous permettant d'y accéder depuis l'extérieur du cluster.
