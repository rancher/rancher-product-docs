`kubectl` est un utilitaire CLI pour exécuter des commandes sur des clusters Kubernetes. Il est nécessaire pour de nombreuses tâches de maintenance et d'administration dans Rancher 2.x.

[#_installation]
== Installation

Voir https://kubernetes.io/docs/tasks/tools/install-kubectl/[Installation de kubectl] pour l'installation sur votre système d'exploitation.

[#_configuration]
== Configuration

Lorsque vous créez un cluster Kubernetes avec RKE2/K3s, le fichier Kubeconfig est stocké à `/etc/rancher/rke2/rke2.yaml` ou `/etc/rancher/k3s/k3s.yaml` selon la distribution choisie. Ces fichiers sont utilisés pour configurer l'accès au cluster Kubernetes.

Testez votre connectivité avec `kubectl` et vérifiez si vous pouvez obtenir la liste des nœuds.

----
kubectl get nodes

165.227.116.167               Ready     controlplane,etcd,worker   11m       v1.10.1
165.227.127.226               Ready     controlplane,etcd,worker   11m       v1.10.1
