`rke2-selinux` fournit des politiques pour RKE2. Il est installé automatiquement lorsque le script d'installation de RKE2 détecte qu'il fonctionne sur une distribution basée sur RPM.

Le dépôt GitHub `rke2-selinux` est https://github.com/rancher/rke2-selinux[ici.]

Pour plus d'informations sur l'installation de RKE2 sur des hôtes avec SELinux activé, consultez la https://documentation.suse.com/cloudnative/rke2/latest/en/install/methods.html#rpm[documentation de RKE2.]
