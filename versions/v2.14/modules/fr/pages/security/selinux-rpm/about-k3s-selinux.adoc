`k3s-selinux` fournit des politiques pour K3s. Il est installé automatiquement lorsque le script d'installation de K3s détecte qu'il fonctionne sur une distribution basée sur RPM.

Le dépôt GitHub `k3s-selinux` est https://github.com/rancher/k3s-selinux[ici].

Pour plus d'informations sur l'installation de K3s sur des hôtes avec SELinux activé, consultez la https://documentation.suse.com/cloudnative/k3s/latest/en/advanced.html#_selinux_support[documentation de K3s].
