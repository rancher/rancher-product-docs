. Dans le coin supérieur gauche, cliquez sur *☰ > Gestion des clusters*.
. Sur la page *Clusters*, allez au cluster où vous souhaitez installer Conformité et cliquez sur *Explorer*.
. Dans la barre de navigation de gauche, cliquez sur *Applis > Charts*.
. Cliquez sur *Conformité*.
. Cliquez sur *Installer*.

*Résultat :* L'application d'analyse de conformité est déployée sur le cluster Kubernetes.
