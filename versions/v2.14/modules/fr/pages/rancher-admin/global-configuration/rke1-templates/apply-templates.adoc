Vous pouvez créer un cluster à partir d'un modèle RKE que vous avez créé, ou à partir d'un modèle qui a été {xref-access-share-rke1-templates}[partagé avec vous.]

Les modèles RKE peuvent être appliqués à de nouveaux clusters.

Vous pouvez <<_converting_an_existing_cluster_to_use_an_rke_template,enregistrer la configuration d'un cluster existant en tant que modèle RKE.>> Ensuite, les paramètres du cluster ne peuvent être modifiés que si le modèle est mis à jour.

Vous ne pouvez pas changer un cluster pour utiliser un modèle RKE différent. Vous ne pouvez mettre à jour le cluster qu'à une nouvelle révision du même modèle.

[#_creating_a_cluster_from_an_rke_template]
== Créer un cluster à partir d'un modèle RKE

Pour ajouter un cluster {xref-launch-k8s-with-rancher}[hébergé par un fournisseur d'infrastructure] en utilisant un modèle RKE, suivez ces étapes{nbsp}:

. Dans le coin supérieur gauche, cliquez sur *☰ > Gestion des clusters*.
. Sur la page *Clusters*, cliquez sur *Créer* et choisissez le fournisseur d'infrastructure.
. Fournissez le nom du cluster et les détails du modèle de nœud comme d'habitude.
. Pour utiliser un modèle RKE, sous les *Options de Cluster*, cochez la case pour *Utiliser un modèle RKE existant et une révision*.
. Choisissez un modèle RKE et une révision dans le menu déroulant.
. Facultatives : Vous pouvez modifier tous les paramètres que le propriétaire du modèle a marqués comme *Autoriser la substitution par l'utilisateur* lors de la création du modèle. S'il y a des paramètres que vous souhaitez changer, mais que vous n'avez pas l'option de le faire, vous devrez contacter le propriétaire du modèle pour obtenir une nouvelle révision du modèle. Ensuite, vous devrez modifier le cluster pour le mettre à niveau vers la nouvelle révision.
. Cliquez sur *Créer* pour lancer le cluster.

[#_updating_a_cluster_created_with_an_rke_template]
== Mise à jour d'un cluster créé avec un modèle RKE

Lorsque le propriétaire du modèle crée un modèle, chaque paramètre dispose d'un interrupteur dans l'interface utilisateur de Rancher qui indique si les utilisateurs peuvent remplacer le paramètre.

* Si le paramètre permet un remplacement par l'utilisateur, vous pouvez mettre à jour ces paramètres dans le cluster en xref:[modifiant le cluster.]
* Si l'interrupteur est désactivé, vous ne pouvez pas modifier ces paramètres à moins que le propriétaire du cluster ne crée une révision du modèle qui vous permet de les remplacer. S'il y a des paramètres que vous souhaitez changer, mais que vous n'avez pas l'option de le faire, vous devrez contacter le propriétaire du modèle pour obtenir une nouvelle révision du modèle.

Si un cluster a été créé à partir d'un modèle RKE, vous pouvez modifier le cluster pour le mettre à jour vers une nouvelle révision du modèle.

Les paramètres d'un cluster existant peuvent être <<_converting_an_existing_cluster_to_use_an_rke_template,enregistrés en tant que modèle RKE.>> Dans cette situation, vous pouvez également modifier le cluster pour mettre à jour le cluster vers une nouvelle révision du modèle.

[NOTE]
====

Vous ne pouvez pas changer le cluster pour utiliser un modèle RKE différent. Vous ne pouvez mettre à jour le cluster qu'à une nouvelle révision du même modèle.
====


[#_converting_an_existing_cluster_to_use_an_rke_template]
== Conversion d'un cluster existant pour utiliser un modèle RKE

Cette section décrit comment créer un modèle RKE à partir d'un cluster existant.

Les modèles RKE ne peuvent pas être appliqués aux clusters existants, sauf si vous enregistrez les paramètres d'un cluster existant en tant que modèle RKE. Cela exporte les paramètres du cluster en tant que nouveau modèle RKE et lie également le cluster à ce modèle. Le résultat est que le cluster ne peut être modifié que si le {xref-manage-rke1-templates}#_mise_à_jour_d_un_modèle[modèle est mis à jour,] et le cluster est mis à niveau vers {xref-manage-rke1-templates}#_mise_à_niveau_d_un_cluster_pour_utiliser_une_nouvelle_révision_du_modèle[utiliser une version plus récente du modèle.]

Pour convertir un cluster existant pour utiliser un modèle RKE,

. Dans le coin supérieur gauche, cliquez sur *☰ > Gestion des clusters*.
. Sur la page *Clusters*, allez au cluster qui sera converti pour utiliser un modèle RKE. Cliquez sur *⋮  > Enregistrer en tant que modèle RKE*.
. Entrez un nom pour le modèle dans le formulaire qui apparaît, puis cliquez sur *Créer*.

*Résultats :*

* Un nouveau modèle RKE est créé.
* Le cluster est converti pour utiliser le nouveau modèle.
* De nouveaux clusters peuvent être <<_creating_a_cluster_from_an_rke_template,créés à partir du nouveau modèle.>>
