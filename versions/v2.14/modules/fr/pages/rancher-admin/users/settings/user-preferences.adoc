Vous pouvez définir des préférences pour personnaliser votre expérience Rancher. Pour modifier les paramètres de préférence{nbsp}:

. Cliquez sur votre avatar utilisateur dans le coin supérieur droit.
. Cliquez sur *Préférences*.

[#_language]
== Langue

Choisissez votre langue pour l'interface utilisateur de Rancher. Les options incluent :

* Anglais
* 简体中文 (Chinois simplifié)

[#_theme]
== Thème

Choisissez votre couleur de fond pour l'interface utilisateur de Rancher. Si vous choisissez *Auto*, la couleur de fond passe de clair à sombre à 18 h, puis revient à 6 h.

[#_login_landing_page]
== Page d'accueil de connexion

Choisissez la page affichée après la connexion. Les options incluent :

* La page d'accueil.
* La zone que vous avez visitée en dernier.
* Un cluster spécifique de votre choix.

[#_display_settings]
== Paramètres d'affichage

Choisissez comment certaines informations sont affichées{nbsp}:

* Format de la date
* Format de l'heure
* Lignes de tableau par page
* Nombre de clusters à afficher dans le menu latéral

[#_confirmation_setting]
== Paramètre de confirmation

Choisissez si vous souhaitez demander une confirmation lors de la réduction des pools de nœuds.

[#_advanced_features]
== Fonctions évoluées

* Activez "Afficher dans l'API".
* Afficher les espaces de noms système gérés par Rancher (non destinés à être modifiés ou supprimés).
* Activez le raccourci clavier pour basculer entre le thème sombre et clair (shift+T).
* Masquer toutes les descriptions de type.
* Activez les fonctionnalités de développement d'extension.

[#_helm_charts]
== Les Helm charts

Choisissez d'afficher uniquement les charts Helm publiés ou d'inclure également les versions préliminaires. Une version est considérée comme préliminaire si elle respecte https://semver.org/#spec-item-9[la spécification] définie par https://semver.org/[Semantic Versioning 2.0.0]. Par exemple, un chart Helm avec une version de `0.1.3-dev.12ab4f` nécessite que `Include Prerelease Versions` soit sélectionné pour être affiché.
