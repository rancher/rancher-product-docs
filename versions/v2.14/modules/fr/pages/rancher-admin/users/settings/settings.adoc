Dans Rancher, chaque utilisateur a un certain nombre de paramètres associés à sa connexion{nbsp}: préférences personnelles, clés API, etc. Vous pouvez configurer ces paramètres en sélectionnant le *Menu des paramètres utilisateur*. Vous pouvez ouvrir ce menu en cliquant sur votre avatar, situé dans le menu principal.

image::user-settings.png[Menu des paramètres utilisateur]

Les paramètres utilisateur disponibles sont{nbsp}:

* {xref-user-api-keys}[API & Clés]{nbsp}: Si vous souhaitez interagir avec Rancher de manière programmatique, vous avez besoin d'une clé API. Suivez les instructions dans cette section pour obtenir une clé.
* {xref-cloud-credentials}[Identifiants Cloud]{nbsp}: Gérez les identifiants cloud utilisés par les pools de machines pour {xref-launch-k8s-with-rancher}[provisionner des nœuds pour des clusters].
* {xref-user-preferences}[Préférences]{nbsp}: Définit des préférences superficielles pour l'interface utilisateur de Rancher.
* Déconnexion{nbsp}: Met fin à votre session utilisateur.
