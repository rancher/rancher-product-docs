La création ou l'association d'identifiants cloud fait partie du processus de création de cluster. Les informations ci-dessous fournissent des conseils sur la gestion des identifiants dans Rancher.

Les identifiants cloud sont liés au profil utilisateur de leur créateur. Ils *ne peuvent pas* être partagés entre des utilisateurs non administrateurs. Cependant, les administrateurs peuvent voir et gérer les identifiants cloud des autres utilisateurs.

[#_creating_a_cloud_credential]
== Créer un identifiant cloud

. Cliquez sur *☰ > Gestion des clusters*.
. Cliquez sur *Identifiants Cloud*.
. Cliquez sur *Create*.
. Cliquez sur un type d'identifiant cloud. Les valeurs de ce menu déroulant sont basées sur les `active` {xref-manage-node-drivers}[pilotes de nœuds] dans Rancher.
. Entrez un nom pour l'identifiant cloud.
. En fonction du type d'identifiant cloud sélectionné, saisissez les valeurs requises pour vous authentifier auprès du fournisseur d'infrastructure.
. Cliquez sur *Create*.

**Résultat :** L'identifiant cloud est créé.

[#_updating_a_cloud_credential]
== Mettre à jour un identifiant cloud

Lorsque les identifiants d'accès sont modifiés ou compromis, la mise à jour d'un identifiant cloud vous permet de remplacer ces identifiants tout en conservant le même modèle de nœud.

. Cliquez sur *☰ > Gestion des clusters*.
. Cliquez sur *Identifiants Cloud*.
. Choisissez l'identifiant cloud que vous souhaitez modifier et cliquez sur le *⋮ > Modifier la configuration*.
. Mettez à jour les informations d'identification et cliquez sur *Enregistrer*.

**Résultat :** L'identifiant cloud est mis à jour avec les nouveaux identifiants d'accès.

[#_deleting_a_cloud_credential]
== Supprimer un identifiant cloud

. Cliquez sur *☰ > Gestion des clusters*.
. Cliquez sur *Identifiants Cloud*.
. Vous pouvez soit supprimer individuellement un identifiant cloud, soit effectuer une suppression en masse.
 ** Pour supprimer individuellement un identifiant cloud, choisissez l'identifiant cloud que vous souhaitez modifier et cliquez sur le *⋮ > Supprimer*.
 ** Pour supprimer des identifiants cloud en masse, sélectionnez un ou plusieurs identifiants cloud dans la liste. Cliquez sur *Supprimer*.
. Confirmez que vous souhaitez supprimer ces identifiants cloud.
