L'authentification locale est la valeur par défaut jusqu'à ce que vous configuriez un fournisseur d'authentification externe. Rancher stocke les informations des comptes utilisateurs, telles que les noms d'utilisateur et les mots de passe, localement. Par défaut, l'utilisateur `admin` qui se connecte à Rancher pour la première fois est un utilisateur local.

[#_adding_local_users]
== Ajouter des utilisateurs locaux

Que vous utilisiez ou non l'authentification externe, vous devriez créer quelques utilisateurs d'authentification locale afin de pouvoir continuer à utiliser Rancher si votre service d'authentification externe rencontre des problèmes.

. Dans le coin supérieur gauche, cliquez sur *☰ > Utilisateurs et authentification*.
. Dans le menu de navigation à gauche, cliquez sur *Utilisateurs*.
. Cliquez sur *Create*.
. Complétez le formulaire *Ajouter un utilisateur*.
. Cliquez sur *Create*.
