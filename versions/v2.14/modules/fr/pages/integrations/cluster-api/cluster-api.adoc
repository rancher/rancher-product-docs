https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name}] est un https://kubernetes.io/docs/concepts/extend-kubernetes/operator/#operators-in-kubernetes[Opérateur Kubernetes] qui gère le cycle de vie des clusters Kubernetes provisionnés, en fournissant une intégration entre votre Cluster API (CAPI) et Rancher. Avec Rancher Turtles, vous pouvez{nbsp}:

* Importer des clusters CAPI dans Rancher, en installant l'Agent de Cluster Rancher dans les clusters provisionnés par CAPI.
* Gérer l'https://cluster-api-operator.sigs.k8s.io/[Opérateur CAPI] en utilisant la ressource {url-turtles-capiprovider}[CAPIProvider].

La section {xref-turtles-overview}[Aperçu] décrit les options d'installation, l'architecture de Rancher Turtles et une brève démonstration. Pour plus de détails, consultez la https://documentation.suse.com/cloudnative/cluster-api/{turtles-docs-version}/en/index.html[{turtles-product-name} documentation].
