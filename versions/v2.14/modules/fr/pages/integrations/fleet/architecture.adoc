Fleet peut gérer les déploiements depuis git de YAML Kubernetes brut, de charts Helm, de Kustomize ou de toute combinaison des trois. Quelle que soit la source, toutes les ressources sont dynamiquement transformées en charts Helm, et Helm est utilisé comme moteur pour déployer tout dans le cluster. Cela vous donne un haut degré de contrôle, de cohérence et d'auditabilité. Fleet se concentre non seulement sur la capacité à monter en charge, mais aussi sur l'offre d'un haut degré de contrôle et de visibilité sur ce qui est installé dans le cluster.
