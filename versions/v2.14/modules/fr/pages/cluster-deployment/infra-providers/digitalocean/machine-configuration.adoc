Pour plus de détails sur DigitalOcean, Droplets, consultez la https://docs.digitalocean.com/products/compute/[documentation officielle].

[#_region]
== Région

Configurez la https://docs.digitalocean.com/glossary/region/[région] où les Droplets sont créés.

[#_size]
== Taille

Configurez la https://docs.digitalocean.com/products/droplets/resources/choose-plan/[taille] des Droplets.

[#_os_image]
== Image du système d’exploitation

Configurez l’ https://docs.digitalocean.com/products/images/[image] du système d’exploitation à partir de laquelle les Droplets sont créés.

[#_monitoring]
== Surveillance

Activez l’agent DigitalOcean pour un https://docs.digitalocean.com/products/monitoring/[suivi] supplémentaire.

[#_ipv6]
== IPv6

Activez l’IPv6 pour les Droplets.

Pour plus d’informations, consultez la https://docs.digitalocean.com/products/networking/ipv6[documentation IPv6 de DigitalOcean].

[#_private_networking]
== Réseautage privé

Activez le réseautage privé pour les Droplets.

[#_droplet_tags]
== Étiquettes de Droplet

Appliquez une étiquette (label) à un Droplet. Les étiquettes ne peuvent contenir que des lettres, des chiffres, des deux-points, des tirets et des underscores. Par exemple : `my_server`.
