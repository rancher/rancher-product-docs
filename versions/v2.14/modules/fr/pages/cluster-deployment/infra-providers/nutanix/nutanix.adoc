https://www.nutanix.com/products/acropolis[Système d'exploitation Nutanix Acropolis] (Nutanix AOS) est un système d'exploitation pour la plateforme d'infrastructure hyper-convergée Nutanix. AOS est livré avec un hyperviseur intégré appelé https://www.nutanix.com/products/ahv[Hyperviseur Acropolis], ou AHV. En utilisant Rancher avec Nutanix AOS (AHV), vous pouvez apporter des opérations cloud sur site.

Rancher peut provisionner des nœuds dans AOS (AHV) et y installer Kubernetes. Lors de la création d'un cluster Kubernetes dans AOS, Rancher provisionne d'abord le nombre spécifié de machines virtuelles en communiquant avec l'API Prism Central. Ensuite, il installe Kubernetes sur les machines virtuelles.

Un cluster Nutanix peut se composer de plusieurs groupes de machines virtuelles avec des propriétés distinctes, telles que la quantité de mémoire ou le nombre de vCPUs. Ce regroupement permet un contrôle précis sur la taille des nœuds pour chaque rôle Kubernetes.
