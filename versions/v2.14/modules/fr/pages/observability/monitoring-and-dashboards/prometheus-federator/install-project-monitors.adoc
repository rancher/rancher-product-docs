Installez *moniteurs de projet* dans chaque projet où vous souhaitez activer la surveillance de projet.

. Cliquez sur *☰ > Gestion des clusters*.
. Sur la page *Clusters*, allez au cluster où vous souhaitez activer la surveillance de projet et cliquez sur *Explorer*.
. Cliquez sur menu:Surveillance[moniteurs de projet] dans la barre de navigation à gauche. Puis cliquez sur *Créer* en haut à droite.
+
image::project-monitors.png[Moniteurs de projet]

. Sélectionnez votre projet dans le menu déroulant, puis cliquez à nouveau sur *Créer*.
+
image::create-project-monitors.png[Créer des Moniteurs de projet]
