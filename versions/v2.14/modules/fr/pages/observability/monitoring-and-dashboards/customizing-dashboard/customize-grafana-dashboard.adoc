Dans cette section, vous apprendrez à personnaliser le tableau de bord Grafana pour afficher des métriques qui s'appliquent à un certain conteneur.

[#_prerequisites]
== Conditions préalables

Avant de pouvoir personnaliser un tableau de bord Grafana, l'application `rancher-monitoring` doit être installée.

Pour voir les liens vers les interfaces de surveillance externes, y compris les tableaux de bord Grafana, vous aurez besoin d'au moins un rôle {xref-rancher-monitoring-rbac}#_utilisateurs_avec_permissions_basées_sur_rancher[membre du projet.]

[#_signing_in_to_grafana]
== Connexion à Grafana

. Dans l'interface utilisateur de Rancher, allez au cluster qui contient le tableau de bord que vous souhaitez personnaliser.
. Dans le menu de navigation à gauche, cliquez sur *Surveillance.*
. Cliquez sur *Grafana.* Le tableau de bord Grafana devrait s'ouvrir dans un nouvel onglet.
. Allez à l'icône de connexion dans le coin inférieur gauche et cliquez sur *Se connecter.*
. Connectez-vous à Grafana. Le nom d'utilisateur et le mot de passe par défaut pour l'instance Grafana sont `admin/prom-operator`. (Peu importe qui a le mot de passe, l'autorisation d'administrateur de cluster dans Rancher est toujours requise pour accéder à l'instance Grafana.) Des identifiants alternatifs peuvent également être fournis lors du déploiement ou de la mise à niveau du chart.

[#_getting_the_promql_query_powering_a_grafana_panel]
== Obtenir la requête PromQL alimentant un panneau Grafana

Pour tout panneau, vous pouvez cliquer sur le titre et cliquer sur *Explorer* pour obtenir les requêtes PromQL alimentant le graphique.

Pour cet exemple, nous aimerions obtenir l'utilisation de l'UC pour le conteneur Alertmanager, donc nous cliquons sur menu:Utilisation de l'UC[Inspecter.]

L'onglet *Données* montre les données sous-jacentes sous forme de série temporelle, avec le temps dans la première colonne et le résultat de la requête PromQL dans la deuxième colonne. Copiez la requête PromQL.

----
(1 - (avg(irate({__name__=~"node_cpu_seconds_total|windows_cpu_time_total",mode="idle"}[5m])))) * 100
----

Vous pouvez ensuite modifier la requête dans le panneau Grafana ou créer un nouveau panneau Grafana en utilisant la requête.

Reportez-vous également à la section:

* https://grafana.com/docs/grafana/latest/panels-visualizations/configure-panel-options/#edit-a-panel[Documentation Grafana sur l'édition d'un panneau]
* https://grafana.com/docs/grafana/latest/panels-visualizations/panel-editor-overview[Documentation Grafana sur l'ajout d'un panneau à un tableau de bord]
