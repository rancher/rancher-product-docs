[IMPORTANT]
====
La rotation des clés de chiffrement est activée par défaut et ne peut pas être désactivée.
====

Pour faire pivoter les clés dans la GUI Rancher{nbsp}:

. Cliquez sur *☰ > Gestion des clusters*.
. Sélectionnez *⋮ > Rotation des Clés de Chiffrement* à l'extrême droite de l'écran à côté de votre cluster choisi{nbsp}:
+
image::rke2-encryption-key.png[Rotation des Clés de Chiffrement]

NOTE: Pour plus d'informations sur la configuration de chiffrement des secrets RKE2, veuillez consulter la https://documentation.suse.com/cloudnative/rke2/latest/en/security/secrets_encryption.html[documentation RKE2].
