Rancher prend en charge le stockage persistant avec une variété de plugins de volume. Cependant, avant d'utiliser l'un de ces plugins pour lier le stockage persistant à vos charges de travail, vous devez configurer le stockage lui-même, qu'il s'agisse d'une solution basée sur le cloud d'un fournisseur de services ou d'une solution sur site que vous gérez vous-même.

Pour votre commodité, Rancher propose une documentation sur la façon de configurer certaines des méthodes de stockage populaires{nbsp}:
