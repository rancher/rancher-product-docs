= Integração com marketplaces de nuvem
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-08-05
:page-revdate: {revdate}
:community-path: integrations-in-rancher/cloud-marketplace/index.adoc 
:product-path: installation-and-upgrade/hosted-kubernetes/cloud-marketplace/cloud-marketplace.adoc

O Rancher oferece integração com marketplaces de nuvem para facilitar a compra de suporte para instalações hospedadas em determinados provedores de nuvem. Além disso, essa integração também fornece a capacidade de gerar um pacote de suporte que pode ser fornecido ao suporte do Rancher.

Essa integração suporta apenas AWS.
