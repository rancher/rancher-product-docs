[#_what_can_i_expect_when_a_feature_is_marked_for_deprecation]
== O que posso esperar quando um recurso é marcado para descontinuação?

Na versão em que a funcionalidade é marcada como "Descontinuada", ela ainda estará disponível e suportada, permitindo que se faça o upgrade conforme o procedimento usual. Uma vez feito o upgrade, os usuários/admins devem começar a planejar a migração da funcionalidade descontinuada antes de fazer o upgrade para a versão em que ela foi marcada como removida. A recomendação para novas implantações é não usar o recurso descontinuado.
